# FreeBSD 5.3-RELEASE 发行公告

**日期：** 2004 年 11 月 6 日 16:19:57 -0700
**发件人：** Scott Long <scottl@FreeBSD.org>
**收件人：** <freebsd-announce@FreeBSD.org>
**主题：** [FreeBSD-Announce] FreeBSD 5.3-RELEASE Announcement

我很荣幸地宣布 FreeBSD 5.3-RELEASE 现已发布。此版本标志着 FreeBSD 5.*X* 系列的一个里程碑，也是 5-STABLE 发行分支的起点。自 5.2.1 以来的诸多变更包括：

- 为 i386 平台引入了二进制兼容接口，允许在内核中原生运行 Microsoft Windows NDIS 网络驱动程序。
- 网络和套接字子系统现已实现多线程和可重入。这允许在处理和转发本地及远程网络流量时更好地利用 SMP 并行性。
- 开发环境已更新为 GCC 3.4.2、Binutils 2.15 和 GDB 6.1。
- 图形环境的选择已更新，包含 X.org 6.7、Gnome 2.6.2 和 KDE 3.3.0。

此版本还着重于测试和错误修复，并冻结了大部分内核和用户态 API。我们鼓励用户和供应商考虑迁移到此版本，因为 FreeBSD 5.*X* 不再被视为“新技术”发行版系列。有关从 FreeBSD 4.*X* 迁移到 5.*X* 的信息可在以下地址找到：

[http://www.FreeBSD.org/releases/5.3R/migration-guide.html](http://www.freebsd.org/releases/5.3R/migration-guide.html)

有关新功能和已知问题的完整列表，请参见发行说明和勘误列表：

[http://www.FreeBSD.org/releases/5.3R/relnotes.html](http://www.freebsd.org/releases/5.3R/relnotes.html)
[http://www.FreeBSD.org/releases/5.3R/errata.html](http://www.freebsd.org/releases/5.3R/errata.html)

有关 FreeBSD 发布工程活动的更多信息，请参见：

[http://www.FreeBSD.org/releng](http://www.freebsd.org/releng)

## 可用性

FreeBSD 5.3-RELEASE 支持 i386、pc98、alpha、sparc64、amd64 和 ia64 架构，可使用可引导介质直接通过网络安装或复制到本地 NFS/FTP 服务器。所有架构的发行版现已提供。

请通过从我们的支持厂商处购买介质来继续支持 FreeBSD 项目。以下公司将提供基于 FreeBSD 5.3 的产品：

- FreeBSD Mall, Inc. [http://www.freebsdmall.com/](http://www.freebsdmall.com/)
- Daemonnews, Inc. [http://www.bsdmall.com/freebsd1.html](http://www.bsdmall.com/freebsd1.html)

如果你无法负担介质形式的 FreeBSD、不耐烦等待，或仅想用于推广目的，那么请务必下载 ISO 镜像。我们无法保证所有镜像站点都提供较大的 ISO 镜像，但至少可从以下站点获取。发行版镜像的 MD5 校验和附在本消息末尾。
