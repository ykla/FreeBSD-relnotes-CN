# FreeBSD 5.0-RELEASE 发行公告

**日期：** 2003 年 1 月 19 日 10:03:42 -0700
**发件人：** <scottl@FreeBSD.org> (Scott Long)
**收件人：** <freebsd-announce@FreeBSD.org>
**主题：** FreeBSD 5.0 Release Available

我很荣幸地宣布 FreeBSD 5.0-RELEASE 现已发布。经过近三年的工作积累，此版本使 FreeBSD 踏上了先进多处理器和应用程序线程支持的道路，并引入了对 sparc64 和 ia64 平台的支持。新增和改进的功能包括：

- UFS2，第二代 UFS 文件系统，打破了当前 1TB 文件系统的限制。
- 后台文件系统检查（bgfsck）和文件系统快照消除了进行文件系统修复和备份任务时的停机需求。
- 对强制访问控制（MAC）的实验性支持为管理员定义系统安全策略提供了可扩展和灵活的手段。
- 内核中的细粒度锁定为多处理器系统更高效率铺平了道路。
- 对蓝牙、ACPI、CardBus、IEEE 1394 以及实验性硬件加密加速的支持使 FreeBSD 保持在新技术的最前沿。
- GCC 3.2.1 编译器提供了不断改进的 GNU 编译器集合的最新版本。
- GEOM，可扩展和灵活的存储框架，以及 DEVFS，设备虚拟文件系统，简化了存储和设备管理，同时为新的企业存储技术打开了大门。
- 对 sparc64 和 ia64 平台的支持扩展了 FreeBSD 对先进 64 位计算平台的支持。

尽管 FreeBSD 5.0 包含许多令人兴奋的新功能，但它可能不适合所有用户。较为保守的用户可能更愿意继续使用 FreeBSD 4.X。有关各种权衡的信息，以及 FreeBSD 4.X 和 5.X 的未来计划说明，可在早期采用者指南中找到：

## 可用性

FreeBSD 5.0-RELEASE 支持 i386、pc98、alpha、sparc64 和 ia64 架构，可使用引导软盘直接通过网络安装或复制到本地 NFS/FTP 服务器。所有架构的发行版现已提供。

请通过从我们的支持厂商处购买介质来继续支持 FreeBSD 项目。以下公司将提供基于 FreeBSD 5.0 的产品：
