# FreeBSD 5.0-RELEASE 安装说明

FreeBSD 的安装说明针对不同平台进行了定制，因为安装 FreeBSD 的过程高度依赖于硬件平台。

FreeBSD 5.0-RELEASE 的安装说明可在以下平台获取：

- [alpha](5.0-alpha-installation.md)
- [i386](5.0-i386-installation.md)
- [ia64](5.0-ia64-installation.md)
- [pc98](5.0-pc98-installation.md)
- [sparc64](5.0-sparc64-installation.md)

当前正在开发的所有平台列表可在 [支持的平台](https://www.freebsd.org/platforms/) 页面上找到。
