# FreeBSD 4.1-RELEASE 发行公告

**日期：** Thu, 27 Jul 2000 05:17:09 -0700
**发件人：** “Jordan K. Hubbard” <jkh@zippy.osd.bsdi.com>
**收件人：** announce@FreeBSD.org
**主题：** 4.1-RELEASE now available from ftp.freebsd.org

我非常高兴地宣布 FreeBSD 4.1-RELEASE 发布，这是 4.x-STABLE 分支技术的最新版本。继 2000 年 3 月 FreeBSD 4.0 发布之后，修复了许多错误，处理了重要的安全问题，并新增了不少新功能。更多信息请参见发行说明。

4.1-RELEASE 现已面向 [i386](ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/i386/4.1-RELEASE) 和 [alpha](ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/alpha/4.1-RELEASE) 架构提供，可使用引导软盘直接通过网络安装，或复制到本地 NFS/FTP 服务器。ISO 镜像稍后也会提供（见下文）。

## ISO（CD）镜像

安装 CD 的 ISO 镜像将于 2000 年 8 月 1 日前提供，在此之前这些文件还需进行更多集成测试。鉴于 ISO 镜像体积庞大（每个约 650MB），且多数人不愿重复传输如此大的文件，这额外的等待是必要的。ISO 镜像就位后将发送后续公告，因此请不要再发电子邮件询问它们在哪里或何时就绪。一旦准备就绪，就会上传并发送公告。

我们也无法保证所有镜像站点都会提供这些大型 ISO 镜像，但至少可从以下地址获取：

**ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/i386/ISO-IMAGES/4.1-install.iso**

**ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/alpha/ISO-IMAGES/4.1-install.iso**

如果你买不起 CD、等不及，或只是想用于推广目的，那么尽管下载 ISO；否则请继续通过向 BSDi 购买其官方 CD 发行版来支持 FreeBSD 项目。FreeBSD 4.1-RELEASE 可向 [The FreeBSD Mall](http://www.freebsdmall.com) 订购 4 CD 套装，不久后即可发货。每套 CD 包含 x86 或 alpha 架构的 FreeBSD 安装文件及应用软件包（每种架构有其独立的 CD 套装）。如需用于构建 Ports 集合中 Ports 的 distfiles 集合，另请参见 FreeBSD Toolkit，这是一套 6 CD 套装，包含所有无法再放入 4 CD 套装的额外文件。你也可以通过电话、邮政信件、传真或电子邮件订购：
