# FreeBSD 6.2-RELEASE 发行公告

**日期：** 2007 年 1 月 15 日星期一 00:29:19 -0500
**发件人：** Ken Smith [kensmith@FreeBSD.org](mailto:kensmith@FreeBSD.org)
**收件人：** [freebsd-announce@FreeBSD.org](mailto:freebsd-announce@FreeBSD.org)
**主题：** [FreeBSD-Announce] FreeBSD 6.2 已发布

FreeBSD 发布工程团队很高兴地宣布 FreeBSD 6.2-RELEASE 现已发布。此版本延续了 6-STABLE 分支的开发，提供了性能和稳定性改进、许多 bug 修复和新功能。其中一些亮点：

- [freebsd-update(8)](https://man.freebsd.org/cgi/man.cgi?query=freebsd-update&sektion=8&manpath=FreeBSD+6.2-RELEASE) 为安全修复和勘误补丁提供官方支持的二进制更新
- 实验性的 CAPP 安全事件审计支持
- OpenBSM 审计命令行工具套件和库
- KDE 更新至 3.5.4，GNOME 更新至 2.16.1
- 集成了 [csup(1)](https://man.freebsd.org/cgi/man.cgi?query=csup&sektion=1&manpath=FreeBSD+6.2-RELEASE) CVSup 客户端
- 向 [geli(8)](https://man.freebsd.org/cgi/man.cgi?query=geli&sektion=8&manpath=FreeBSD+6.2-RELEASE) 添加了磁盘完整性保护和认证
- 新增 [amdsmb(4)](https://man.freebsd.org/cgi/man.cgi?query=amdsmb&sektion=4&manpath=FreeBSD+6.2-RELEASE)、[enc(4)](https://man.freebsd.org/cgi/man.cgi?query=enc&sektion=4&manpath=FreeBSD+6.2-RELEASE)、[ipmi(4)](https://man.freebsd.org/cgi/man.cgi?query=ipmi&sektion=4&manpath=FreeBSD+6.2-RELEASE)、[nfsmb(4)](https://man.freebsd.org/cgi/man.cgi?query=nfsmb&sektion=4&manpath=FreeBSD+6.2-RELEASE)、[stge(4)](https://man.freebsd.org/cgi/man.cgi?query=stge&sektion=4&manpath=FreeBSD+6.2-RELEASE) 驱动程序
- [ipfw(4)](https://man.freebsd.org/cgi/man.cgi?query=ipfw&sektion=4&manpath=FreeBSD+6.2-RELEASE) 数据包标记
- Linux 仿真支持 sysfs
- BIND 更新至 9.3.3
- 许多驱动程序更新，包括 [em(4)](https://man.freebsd.org/cgi/man.cgi?query=em&sektion=4&manpath=FreeBSD+6.2-RELEASE)、[arcmsr(4)](https://man.freebsd.org/cgi/man.cgi?query=arcmsr&sektion=4&manpath=FreeBSD+6.2-RELEASE)、[ath(4)](https://man.freebsd.org/cgi/man.cgi?query=ath&sektion=4&manpath=FreeBSD+6.2-RELEASE)、[bce(4)](https://man.freebsd.org/cgi/man.cgi?query=bce&sektion=4&manpath=FreeBSD+6.2-RELEASE)、[ata(4)](https://man.freebsd.org/cgi/man.cgi?query=ata&sektion=4&manpath=FreeBSD+6.2-RELEASE) 和 [iwi(4)](https://man.freebsd.org/cgi/man.cgi?query=iwi&sektion=4&manpath=FreeBSD+6.2-RELEASE)

## 可用性

FreeBSD 6.2-RELEASE 现已为 alpha、amd64、i386、ia64、pc98、powerpc 和 sparc64 架构提供。它可从可引导 ISO 镜像安装或通过网络安装；所需文件可通过 FTP 或 BitTorrent 下载，具体见下文所述。虽然某些较小的 FTP 镜像可能不包含所有架构，但通常都会包含较常见的架构，如 i386 和 amd64。

发行版 ISO 镜像的 MD5 和 SHA256 哈希值附在本消息末尾。

大多数架构的发行版 ISO 镜像内容已发生变化。以 i386 架构为例，存在名为“bootonly”、“disc1”、“disc2”和“docs”的 ISO 镜像。“bootonly”镜像适用于引导机器进行基于 FTP 或 NFS 的网络安装。“disc1”和“disc2”镜像用于进行包含基本软件包集合的完整安装，安装过程中无需访问 FTP 或 NFS 服务器。此外，“disc1”支持引导进入“基于 live CD 的文件系统”和系统救援模式。“docs”镜像包含所有支持语言的所有文档。大多数用户会发现仅需“disc1”和“disc2”即可。

FreeBSD 6.2-RELEASE 也可从多家供应商处购买 CD-ROM 版本。将提供 FreeBSD 6.2 相关产品的两家供应商是：

## FreeBSD Update

[freebsd-update(8)](https://man.freebsd.org/cgi/man.cgi?query=freebsd-update&sektion=8&manpath=FreeBSD+6.2-RELEASE) 工具是 FreeBSD Update 二进制更新系统的客户端部分，现已包含在 FreeBSD 基本系统中，并由 FreeBSD 安全团队支持，将为 i386 和 amd64 平台构建二进制安全更新和勘误更新。

此外，[freebsd-update(8)](https://man.freebsd.org/cgi/man.cgi?query=freebsd-update&sektion=8&manpath=FreeBSD+6.2-RELEASE) 工具的实验版本支持在 FreeBSD 发行版之间升级系统。建议 FreeBSD 6.1 系统管理员测试此功能，并在 freebsd-stable 邮件列表上报告遇到的任何问题。更多信息，请参见：

[http://www.daemonology.net/blog/2006-11-26-freebsd-6.1-to-6.2-binary-upgrade.html](http://www.daemonology.net/blog/2006-11-26-freebsd-6.1-to-6.2-binary-upgrade.html)
