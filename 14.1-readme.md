# FreeBSD 14.1-RELEASE 自述文件

## 摘要

本文档简要介绍 FreeBSD 14.1-RELEASE。
内容包括如何获取 FreeBSD、联系 FreeBSD 项目的各种途径，以及其他信息来源的指引。

## 引言

本发行版是 FreeBSD 14.1-RELEASE，即 14-STABLE 分支上的最新版本。

## 关于 FreeBSD

FreeBSD 是基于 4.4 BSD Lite2 衍生的开源操作系统，运行于 64 位 x86 架构 PC 硬件（amd64）、32 位 x86 架构 PC 硬件（i386）、64 位 Arm(R) ARMv8-A AArch64 系统（aarch64）、32 位 Arm(R) ARMv6/7 系统（armv6/7）、PowerPC(R) 系统（powerpc、powerpc64）以及 RISC-V（riscv64）系统。

FreeBSD 兼容多种外设和配置，可用于软件开发、游戏、互联网服务提供等各类场景。

本版本 FreeBSD 包含运行此类系统所需的全部内容，包括内核和基础发行版中所有实用程序的完整源代码。
安装源代码发行版后，可通过一条命令从零开始重新编译整个系统，非常适合学生、研究人员或希望了解系统运作原理的用户。

FreeBSD 还提供大量第三方移植软件（Ports Collection），便于获取和安装各类经典 UNIX(R) 实用程序。
每个 Port 由一组脚本组成，通过一条命令即可获取、配置、编译并安装软件。
超过 30,000 个 Port，涵盖编辑器、编程语言、图形应用等，使 FreeBSD 成为功能强大且全面的操作环境，远超许多商业 UNIX(R) 版本所提供的内容。
多数 Port 还提供预编译的 package，可通过 [pkg(8)](https://man.freebsd.org/cgi/man.cgi?query=pkg&sektion=8&format=html) 实用程序快速安装。

## 目标受众

本版本 FreeBSD 适合所有用户。
经过测试和质量保证阶段，确保了最高的可靠性和稳定性。

## 获取 FreeBSD

可通过多种方式获取 FreeBSD。
最便捷的方式是下载发行版镜像。
最新发行版的镜像链接可在 FreeBSD 网站的 [下载页面](https://www.freebsd.org/where/) 找到。
该页面包含各类镜像的说明以及获取 FreeBSD 副本的其他方式。

## 联系 FreeBSD 项目

### 电子邮件和邮件列表

一般问题或技术支持请发送至 [freebsd-questions 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-questions)。

如果你跟踪 `14-STABLE` 分支：必须订阅 [freebsd-stable 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-stable)，以及时了解可能影响系统使用和维护的开发动态和变更。

FreeBSD 项目主要由志愿者推动，始终欢迎有意协助的人士加入——待实现的改进需求远超现有时间所能完成的数量。
如需就技术问题联系开发者或提供帮助，请发送至 [freebsd-hackers 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-hackers)。

上述列表可能有较大的邮件流量。
如果你的邮件访问速度较慢或费用较高，或仅关注重大公告，可订阅 [freebsd-announce 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-announce)。

所有公开邮件列表均欢迎任何人士加入。
超过一百个邮件列表是公开的；<https://lists.freebsd.org/> 列出了全部列表，并提供浏览存档、订阅和退订的相关信息。

### 提交问题报告

欢迎提交建议、缺陷报告和代码贡献。
如发现任何问题，请勿犹豫，及时报告。
附带修复的缺陷报告更受欢迎。

在有互联网连接的机器上，提交缺陷报告的首选方式是使用 [Bugzilla 缺陷跟踪系统](https://bugs.freebsd.org)。
以此方式提交的问题报告（PR）将被归档并跟踪其进展；FreeBSD 开发者会尽快响应所有报告的缺陷。
FreeBSD 网站提供 [所有活跃 PR 的列表](https://bugs.freebsd.org/search/)；该列表有助于了解其他用户遇到的潜在问题。

更多信息请参阅 FreeBSD 网站上的 [《编写 FreeBSD 问题报告》](https://docs.freebsd.org/en/articles/problem-reports/)，其中包含编写和提交有效问题报告的诸多实用建议。

## 延伸阅读

关于 FreeBSD 的信息来源众多；部分包含在本发行版中，部分可在网上或印刷版本中获取。

### 发行文档

另有若干文件提供关于本发行版的更具体信息。
这些文件以多种格式提供。
多数发行版会同时包含 ASCII 文本（**.TXT**）和 HTML（**.HTM**）版本。
部分发行版还可能包含其他格式，如便携文档格式（**.PDF**）。

* **README.TXT**：本文件，提供 FreeBSD 的概览以及获取发行版的简要说明。
* **RELNOTES.TXT**：发行说明，展示 FreeBSD 14.1-RELEASE 相对于上一版本（FreeBSD 14.0-RELEASE）的新增内容和变更。
* **HARDWARE.TXT**：硬件兼容列表，展示已测试并确认可用的设备。
* **ERRATA.TXT**：发行勘误。发行后期的最新信息可在此文件中找到，主要适用于发行版（而非快照）。在安装 FreeBSD 发行版前，务必查阅此文件，因为它包含自发行版创建以来发现并修复的问题的最新信息。

安装过程中，可通过文档菜单查阅这些文档。
系统安装完成后，可重新运行 [bsdinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=bsdinstall&sektion=8&format=html) 实用程序再次访问该菜单。

> **注意**：如需了解任何最新消息或发行后的问题，在安装任何发行版前阅读其勘误至关重要。随各发行版附带的勘误文件（通常就在本文件旁）按定义已过时，但互联网上保留有持续更新的副本，应将其视为本版本的当前勘误。这些勘误副本位于 [https://www.FreeBSD.org/releases](https://www.FreeBSD.org/releases)（以及任何保持该位置最新镜像的站点）。

### 手册页

与几乎所有 UNIX(R) 类操作系统一样，FreeBSD 包含一组手册页，可 [在线](https://man.freebsd.org/) 查阅，或使用 [man(1)](https://man.freebsd.org/cgi/man.cgi?query=man&sektion=1&format=html) 命令查阅。

这些手册页提供 FreeBSD 用户可用的各类命令和 API 的信息。

部分手册页专门介绍特定主题。
典型示例包括 [tuning(7)](https://man.freebsd.org/cgi/man.cgi?query=tuning&sektion=7&format=html)（性能调优指南）、[security(7)](https://man.freebsd.org/cgi/man.cgi?query=security&sektion=7&format=html)（FreeBSD 安全简介）和 [style(9)](https://man.freebsd.org/cgi/man.cgi?query=style&sektion=9&format=html)（内核编码风格指南）。

### 书籍和文章

FreeBSD 项目维护的文档包括实用的 *FreeBSD Handbook*，其中包含安装和升级操作系统的详细指南；以及 *FreeBSD FAQ*（常见问题解答）。
[手册](https://docs.freebsd.org/en/books/handbook/) 和 [FAQ](https://docs.freebsd.org/en/books/faq/) 在 [文档门户](https://docs.freebsd.org/) 和镜像站点在线提供。

项目维护的其他书籍和文章更为专业化——涵盖从邮件列表的有效使用、FreeBSD 与其他操作系统的双系统启动，到新提交者的指南等广泛主题。
与手册和 FAQ 一样，这些文档位于文档门户、镜像站点和 **doc** 仓库中。

如需离线 HTML 和 PDF 格式的文档：可安装特定语言的 package，如 `misc/freebsd-doc-en`（`-en` 表示英文），或多语言 package `misc/freebsd-doc-all`。
或者，使用 **doc** 仓库的副本从源代码构建并安装。

关于 FreeBSD 的其他书籍和文档列表可在 FreeBSD 手册的 [参考文献](https://docs.freebsd.org/en/books/handbook/bibliography/) 中找到。
由于 FreeBSD 深植于 UNIX(R) 传统，许多为 UNIX(R) 系统编写的文章和书籍同样适用，其中部分也列于参考文献中。
