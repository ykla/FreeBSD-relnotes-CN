# FreeBSD 2.2-RELEASE 介质变更公告

来自 [jkh@FreeBSD.org](mailto:jkh@FreeBSD.org)：

2.2-RELEASE 将不再支持 4MB 内存或 1.2MB 软盘驱动器的安装——也就是说，仅支持大于 5MB 内存和 1.44MB 软盘驱动器的系统。实际上，建议的最小内存为 8MB。

因此，如果你的配置存在上述限制且无法升级，建议继续使用 2.1.6-RELEASE。该版本支持这两种安装方式，在你可能希望继续运行此类系统的整个生命周期内，其功能应该绰绰有余。

对于由此给部分用户带来的不便，我们深表歉意。但安装介质的空间压力已持续一段时间，这一变更或多或少是不可避免的。我们讨论取消 4MB 安装和 1.2MB 软盘支持已超过一年，能够将其全部保留在一张软盘上，完全是靠一些极其精巧的技巧（你不会想知道的）。既然我们已跨越到 1.44MB/大于 6MB 级别的机器，至少为未来的增强功能腾出了一些空间，同时仍保持在一张软盘上。

请注意，你仍然可以构建在 4MB 系统中舒适运行的内核，只是无法在仅有 4MB 内存的机器上进行安装。例如，如果你试图构建一台 4MB 内存且配置精简的定制机器，可以简单地在另一台专门为此目的设置的机器上制作其磁盘。
