# FreeBSD 2.2-RELEASE 发行公告

我们非常荣幸地宣布发布 FreeBSD 2.2，这是经过漫长的 ALPHA、BETA 和 GAMMA 测试周期后，2.2 分支技术期待已久的首个版本。

FreeBSD 2.2-RELEASE 现已在 [ftp.FreeBSD.org](ftp://ftp.FreeBSD.org/pub/FreeBSD) 及全球各地的 [FTP 镜像站点](https://www.FreeBSD.org/handbook/mirrors.html) 上提供。也可从 [Walnut Creek CDROM](http://www.cdrom.com/) 订购 CD 版本，即将开始发货。

FreeBSD 2.2 相比 2.1.x 版本在功能上有了相当大的飞跃，我们强烈建议所有人阅读[发行说明](https://www.FreeBSD.org/releases/2.2R/notes.html)以查看新功能列表。
