# FreeBSD 2.0-RELEASE 发行公告

**收件人**：<announce@freefall.cdrom.com>
**主题**：2.0 RELEASE 现已在 ftp.freebsd.org 上发布！
**日期**：1994 年 11 月 22 日（星期二）06:35:37 -0800
**发件人**：“Jordan K. Hubbard” <[jkh@freefall.cdrom.com](mailto:jkh@freefall.cdrom.com)>

FreeBSD 项目团队非常荣幸地宣布发布 FreeBSD 2.0；这是一款面向 Intel PC（i386、i486 和 Pentium 级别）的完整 32 位基于 4.4 BSD Lite 的操作系统。

自约 18 个月前首次发布 FreeBSD 1.0 以来，FreeBSD 几乎发生了翻天覆地的变化。我们完成了从 Berkeley 4.4 代码库的全新移植，在 Novell（USL 和 UNIX 的新所有者）的许可下，使系统的法律地位得以明朗化。向 4.4 的移植还引入了大量新功能、文件系统和增强的驱动支持。凭借我们全新的无法律负担的代码库，我们完全有理由相信，在今后相当长的一段时间内，我们都能继续发布高质量的操作系统，而不再受法律问题的困扰！

FreeBSD 2.0 凝聚了近 2 年的心血和一支国际化开发团队投入的数千小时工作量。希望你喜欢它！

FreeBSD 2.0 还具备先进的安装程序，支持从磁带、CD、NFS 或通过 SLIP、以太网或并口的 FTP 安装，以及从 DOS 软盘或硬盘分区安装。我们认为，这是我们迄今为止最易用的安装程序！我们采纳了此前 2.0 ALPHA 版本的大量反馈建议，因此 RELEASE 版本远比当年的 ALPHA 版本安全[希望如此 :-) ]！

FreeBSD 2.0 还支持与其他操作系统更友好地共存，允许你轻松挂载 DOS 文件系统并安装多系统启动管理器，而无需退出安装程序。所有计划的安装方式现均已支持。对于遇到麻烦的用户，启动软盘还附带了故障排除指南，或许能提供帮助。

欲了解 FreeBSD 的新特性或通用功能的更多信息，我们强烈建议你直接下载启动软盘并从中启动。你可以通过简单的菜单阅读发行说明，且不会对硬盘内容造成任何危险（除非你刻意选择“继续安装”，那你的命运就掌握在自己手中了！:-)）。

我们还将很快提供从 ALPHA 到 RELEASE 的升级路径，以方便那些热心帮助我们测试 2.0 早期快照的用户。请关注 announce 邮件列表以获取其发布通知，或者干脆心急一点，直接将 2.0R 的 bindist 解压到你的 ALPHA 版本上！:-)

希望以 CD-ROM 形式获取此版本 2.0 的用户，请联系我们的赞助商 Walnut Creek CDROM（[info@cdrom.com](mailto:info@cdrom.com)）或其他 CD 供应商，他们无疑也会推出各自的发行版本。

如果你当前运行的是 1.x 版本并寻求升级路径，很遗憾，目前仅支持全新安装。只需在从 2.0 介质重新安装前备份好密码和用户文件，然后再恢复即可。如果需求足够大，且我们能找到简便的实现方式，将会提供相应方案。但需要理解的是，1.x 与 2.0 之间的差异**巨大**，能否实现尚不确定。

无法或不愿下载启动软盘的用户，也可通过邮件获取发行说明——发送邮件至 <info@FreeBSD.org> 即可获取自动回复。我们的网页也会持续提供最新信息：

<http://www.freebsd.org>

毫无疑问，各镜像站点在同步后也会提供此版本。不过，ftp.freebsd.org 接入的是 T3 专线并支持 300 个并发用户（它本身就是一台 FreeBSD 机器 :-)），因此在镜像站点同步之前，你从此站点获取应该不会有太大困难。

如果你直接接入互联网，则无需从该站点下载 bindist！只需下载 2 张启动软盘，开始安装并选择 FTP 安装方式——剩下的事情它会透明地为你完成。
