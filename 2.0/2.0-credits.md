# FreeBSD 2.0-RELEASE 贡献者名单

**FreeBSD 2.0**
**贡献者名单**

## 派生软件贡献者

本软件最初派生自 William F. Jolitz 的 386BSD 0.1 版本，但几乎所有的 386BSD 特定代码均不复存在。本软件基本上是在加州大学伯克利分校计算机系统研究组（CSRG）的 4.4 BSD Lite 基础上重新实现的，同时也包含了相关学术贡献者的工作。

其中也融入了 NetBSD 的部分代码，因此我们要感谢所有为 NetBSD 做出贡献的开发者。尽管两个团队之间的关系偶尔有些波折，但双方的目标本质上是一致的：让更多基于 BSD 的操作系统运行在用户的计算机上！我们祝愿 NetBSD 团队在各项事业中取得圆满成功。

## 硬件贡献者

特别感谢 Walnut Creek CDROM 提供了用于开发工作的 486/DX2-66 EISA/VL 系统，更不用说网络接入和其他硬件资源的捐赠。没有他们的支持，完成这一版本是不可能的。

还要感谢 Dermot McDonnell 捐赠了 Toshiba XM3401B CDROM 驱动器，它一直非常有用！
