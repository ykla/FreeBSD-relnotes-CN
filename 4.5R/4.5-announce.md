# FreeBSD 4.5-RELEASE 发行公告

**日期：** 2002 年 1 月 29 日 星期二 20:00:00 -0800
**发件人：** “Murray Stokely” <murray@FreeBSD.org>
**收件人：** <announce@FreeBSD.org>
**主题：** 4.5-RELEASE is now available

我非常高兴地宣布 FreeBSD 4.5-RELEASE 现已发布，这是 FreeBSD -STABLE 开发分支上的最新发行版本。自 2001 年 9 月发布 FreeBSD 4.4 以来，我们进行了数百项修复，更新了许多系统组件，进行了几项重大的性能改进，并处理了各种安全问题。

特别是在网络通信和文件系统方面有显著增强。FreeBSD 4.5 包含对 TCP 协议栈的改进，以提供更好的吞吐量。此外，TCP 性能受益于更大的默认缓冲区大小。最后，FreeBSD 4.5 包含缓解 TCP 拒绝服务攻击影响的新机制。

FFS 文件系统受益于新的目录布局策略，该策略在遍历大型目录结构的操作中展示了显著的性能提升。借助最初由 Apple Computer, Inc. 开发的文件系统测试程序，FFS 和 NFS 代码中的各种 bug 被发现并修复。

全新安装 FreeBSD 的用户应注意新建文件系统的一些更改，旨在改善 FreeBSD 的“开箱即用”性能。特别是 sysinstall(8) 现在为其创建的新文件系统启用 Soft Updates（一种改善磁盘上数据结构性能和可靠性的策略），newfs(8) 程序现在默认创建具有更大块大小的文件系统。

有关此 FreeBSD 发行版最重要更改的更多信息，请参见我们网站的发行版部分：

[http://www.FreeBSD.org/releases/](http://www.freebsd.org/releases/)

在那里你会找到[发行说明](http://www.freebsd.org/releases/4.5R/notes.html)、[硬件兼容列表](http://www.freebsd.org/releases/4.5R/hardware.html)和[勘误](http://www.freebsd.org/releases/4.5R/errata.html)列表。

## 可用性

4.5-RELEASE 提供 i386 和 alpha 架构版本，可使用引导软盘直接通过网络安装，或复制到本地 NFS/FTP 服务器。

我们无法保证所有镜像站点都会提供较大的 ISO 镜像，但至少可从以下地址获取：

- [ftp://ftp.FreeBSD.org/pub/FreeBSD/](ftp://ftp.freebsd.org/pub/FreeBSD/)
- [http://ftp.au.freebsd.org/pub/FreeBSD/](http://ftp.au.freebsd.org/pub/FreeBSD/)
- [ftp://ftp.dk.FreeBSD.org/pub/FreeBSD/](ftp://ftp.dk.freebsd.org/pub/FreeBSD/)
- [ftp://freebsd.nctu.edu.tw/pub/FreeBSD/](ftp://freebsd.nctu.edu.tw/pub/FreeBSD/)

如果你无法负担 CD、急于使用，或仅想用于推广目的，那么请务必下载 ISO 镜像；否则请继续从我们的支持厂商处购买介质以支持 FreeBSD 项目。以下公司对 FreeBSD 的开发做出了重大贡献：

| 公司 | 网站 |
| :--- | :--- |
| FreeBSD Mall, Inc. | [http://www.freebsdmall.com](http://www.freebsdmall.com) |
| FreeBSD Services Ltd. | [http://www.freebsd-services.com](http://www.freebsd-services.com) |
| Daemon News | [http://www.bsdmall.com/freebsd1.html](http://www.bsdmall.com/freebsd1.html) |

每套 CD 包含适用于 i386（“PC”）架构的 FreeBSD 安装程序和应用程序软件包。如需用于构建 Ports 中 port 的一套 distfiles，请参见 FreeBSD Toolkit，这是一套 6 张 CD 的集合，包含无法再容纳于 4 张 CD 套装中的额外内容。

FreeBSD 还可通过以下国家的匿名 FTP 镜像站点获取：阿根廷、澳大利亚、巴西、保加利亚、加拿大、中国、捷克共和国、丹麦、爱沙尼亚、芬兰、法国、德国、香港、匈牙利、冰岛、爱尔兰、以色列、日本、韩国、立陶宛、拉脱维亚、荷兰、波兰、葡萄牙、罗马尼亚、俄罗斯、沙特阿拉伯、南非、斯洛伐克共和国、斯洛文尼亚、西班牙、瑞典、台湾、泰国、乌克兰和英国。

在尝试中央 FTP 站点之前，请先检查你的区域镜像：

**ftp://ftp.\<yourdomain\>.FreeBSD.org/pub/FreeBSD**

任何其他镜像站点将标记为 ftp2、ftp3 等。

有关 FreeBSD 镜像站点的更多信息，请参见 [FreeBSD 手册](http://www.freebsd.org/handbook/mirrors-ftp.html)。

FreeBSD 安装说明最近得到了显著增强。FreeBSD 手册第 2 章（可[在线](http://www.freebsd.org/handbook/install.html)获取）为 FreeBSD 新用户提供了完整的安装指南。
