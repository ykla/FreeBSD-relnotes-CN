# FreeBSD 6.3-RELEASE 发行公告

**日期：** 2008 年 1 月 18 日星期五 14:00:11 -0500
**发件人：** Ken Smith [kensmith@FreeBSD.org](mailto:kensmith@FreeBSD.org)
**收件人：** [freebsd-announce@FreeBSD.org](mailto:freebsd-announce@FreeBSD.org)
**主题：** [FreeBSD-Announce] FreeBSD 6.3 已发布

FreeBSD 发布工程团队很高兴地宣布 FreeBSD 6.3-RELEASE 现已发布。此版本继续 6-STABLE 分支的开发，提供了性能和稳定性改进、大量错误修复以及新功能。其中一些亮点：

- KDE 更新至 3.5.8，GNOME 更新至 2.20.1，Xorg 更新至 7.3
- BIND 更新至 9.3.4
- sendmail 更新至 8.14.2
- 从 OpenBSD/NetBSD 移植了 [lagg(4)](https://man.freebsd.org/cgi/man.cgi?query=lagg&sektion=4&manpath=FreeBSD+6.3-RELEASE) 驱动
- 重新实现了 unionfs 文件系统
- [freebsd-update(8)](https://man.freebsd.org/cgi/man.cgi?query=freebsd-update&sektion=8&manpath=FreeBSD+6.3-RELEASE) 现支持 upgrade 命令

有关新功能和已知问题的完整列表，请参见在线发行说明和勘误列表，地址如下：

- [http://www.FreeBSD.org/releases/6.3R/relnotes.html](http://www.freebsd.org/releases/6.3R/relnotes.html)
- [http://www.FreeBSD.org/releases/6.3R/errata.html](http://www.freebsd.org/releases/6.3R/errata.html)

有关 FreeBSD 发布工程活动的更多信息，请参见：

[http://www.FreeBSD.org/releng/](http://www.freebsd.org/releng/)

FreeBSD 安全团队计划对 6.3-RELEASE 提供支持至 2010 年 1 月 31 日。

## 献辞

FreeBSD 6.3-RELEASE 谨献给 Dr. Jun-ichiro Hagino，他在互联网社区中以 itojun 之名广为人知，以纪念其在 IPv6 协议上富有远见的工作以及对互联网和 BSD 社区的众多其他贡献。

## 可用性

FreeBSD 6.3-RELEASE 现已为 alpha、amd64、i386、pc98 和 sparc64 架构提供。它可以从可引导 ISO 镜像安装或通过网络安装；所需文件可通过 FTP 或 BitTorrent 下载，具体见下文所述。虽然某些较小的 FTP 镜像可能不包含所有架构，但通常都会包含较常见的架构，如 i386 和 amd64。

发行版 ISO 镜像的 MD5 和 SHA256 哈希值附在本消息末尾。

大多数架构的发行版 ISO 镜像内容发生了变化。以 i386 架构为例，存在名为“bootonly”、“disc1”、“disc2”、“disc3”和“docs”的 ISO 镜像。“bootonly”镜像适用于引导机器进行基于 FTP 或 NFS 的网络安装。“disc1”、“disc2”和“disc3”镜像用于进行包含基本软件包集合的完整安装，安装过程中无需访问 FTP 或 NFS 服务器。此外，“disc1”支持引导进入基于 live CD 的文件系统和系统救援模式。“docs”镜像包含所有支持语言的所有文档。大多数用户会发现“disc1”、“disc2”和“disc3”即可满足需求。如果你打算从源码安装 Ports 而非使用发行版附带的预编译软件包，则仅需“disc1”。

也可从多家供应商处购买 CD-ROM 形式的 FreeBSD 6.3-RELEASE。将提供基于 FreeBSD 6.3 的产品的供应商之一是：

- FreeBSD Mall, Inc. [http://www.freebsdmall.com/](http://www.freebsdmall.com/)

## FreeBSD Update

从 FreeBSD 6.3 开始，[freebsd-update(8)](https://man.freebsd.org/cgi/man.cgi?query=freebsd-update&sektion=8&manpath=FreeBSD+6.3-RELEASE) 工具支持对运行早期 FreeBSD 发行版、发行候选和测试版的 i386 和 amd64 系统进行二进制升级。从较旧发行版（特别是早于 6.3-RC1 的版本）升级到 FreeBSD 6.3 的用户需要下载更新版本的 freebsd-update(8) 以支持升级到新发行版。

```sh
# fetch http://people.freebsd.org/~cperciva/freebsd-update-upgrade.tgz
```

强烈建议下载并验证 tarball 的数字签名（由 FreeBSD 安全官的 PGP 密钥签名）。

```sh
# fetch http://people.freebsd.org/~cperciva/freebsd-update-upgrade.tgz.asc
# gpg --verify freebsd-update-upgrade.tgz.asc freebsd-update-upgrade.tgz
```

新的 freebsd-update(8) 可按以下方式解压并运行：

```sh
# tar -xf freebsd-update-upgrade.tgz
# sh freebsd-update.sh -f freebsd-update.conf -r 6.3-RELEASE upgrade
# sh freebsd-update.sh -f freebsd-update.conf install
```

在继续之前，必须使用新安装的内核重启系统。

```sh
# shutdown -r now
```

最后，需要再次运行 freebsd-update.sh 以安装新的用户空间组件，并需要再次重启系统：

```sh
# sh freebsd-update.sh -f freebsd-update.conf install
# shutdown -r now
```

更多信息，请参见：

[http://www.daemonology.net/blog/2007-11-10-freebsd-minor-version-upgrade.html](http://www.daemonology.net/blog/2007-11-10-freebsd-minor-version-upgrade.html)
