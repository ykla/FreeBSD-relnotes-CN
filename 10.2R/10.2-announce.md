# FreeBSD 10.2-RELEASE 发行公告

日期：2015 年 8 月 13 日

FreeBSD 发布工程团队很高兴地宣布 FreeBSD 10.2-RELEASE 现已可用。这是 stable/10 分支的第三个发行版，基于 10.1-RELEASE 的稳定性构建，并引入了新功能。

部分亮点如下：

* **resolvconf(8)** 工具已更新至 3.7.0 版本，改进了 DNS 隐私保护。
* NTP 套件已更新至 4.2.8p3 版本。
* 新增 **rc(8)** 脚本 `growfs`，如果 **/firstboot** 文件存在，该脚本将在引导时调整根文件系统的大小。
* Linux 兼容性版本已更新，支持 Centos(TM) 6 Ports。
* `drm` 代码已更新，匹配 Linux 3.8.13 版本，允许同时运行多个 X 服务器。
* 多项增强和更新，改进了 FreeBSD/arm 支持。
* 多项 ZFS 性能和可靠性改进。
* GNOME 已更新至 3.14.2 版本。
* KDE 已更新至 4.14.3 版本。
* 以及更多...

关于新功能和已知问题的完整列表，请参阅在线发行说明和勘误列表，地址如下：

* <https://www.FreeBSD.org/releases/10.2R/relnotes/>
* <https://www.FreeBSD.org/releases/10.2R/errata/>

关于 FreeBSD 发布工程活动的更多信息，请参阅：

* <https://www.FreeBSD.org/releng/>

## 可用性

FreeBSD 10.2-RELEASE 现已支持 amd64、i386、ia64、powerpc、powerpc64、sparc64 和 armv6 架构。

FreeBSD 10.2-RELEASE 可从可引导 ISO 镜像或通过网络安装。部分架构还支持从 USB 闪存盘安装。所需文件可按以下章节的说明通过 FTP 下载。虽然一些较小的 FTP 镜像可能不包含所有架构，但通常都会包含 amd64 和 i386 等较常见的架构。

本消息底部包含了发行版 ISO、闪存盘和 SD 卡镜像的 SHA256 和 MD5 哈希值。

发行版镜像的 PGP 签名校验和也可在以下地址获取：

* <https://www.FreeBSD.org/releases/10.2R/signatures/>

本公告的 PGP 签名版本可在以下地址获取：

* <https://www.FreeBSD.org/releases/10.2R/announce.asc>

amd64（x86_64）架构还提供了额外的支持 UEFI 的镜像。

作为发行版一部分提供的镜像用途如下：

**dvd1**：
包含安装 FreeBSD 基本操作系统、文档以及一组旨在使图形工作站运行起来的预构建软件包所需的全部内容。它还支持引导进入基于“livefs”的救援模式。如果你能刻录和使用 DVD 大小的介质，这应是你所需的全部。

**disc1**：
包含 FreeBSD 基本操作系统。它还支持引导进入基于“livefs”的救援模式。不包含预构建的软件包。

**bootonly**：
支持使用 CDROM 驱动器引导机器，但不包含从 CD 本身安装 FreeBSD 的安装发行版集合。从 CD 引导后，需要执行基于网络的安装（例如从 FTP 服务器）。

**memstick**：
可写入 USB 闪存盘（闪存驱动器）并用于在能够从 USB 驱动器引导的机器上执行安装。它还支持引导进入基于“livefs”的救援模式。不包含预构建的软件包。

作为如何使用 memstick 镜像的示例，假设 USB 驱动器在你的机器上显示为 **/dev/da0**，类似以下命令应该可行：

```sh
# dd if=FreeBSD-10.2-RELEASE-amd64-memstick.img of=/dev/da0 bs=10240 conv=sync
```

请务必确保目标（of=）正确。

**mini-memstick**：
可写入 USB 闪存盘（闪存驱动器）并用于引导机器，但不包含介质本身的安装发行版集合，类似于 bootonly 镜像。它还支持引导进入基于“livefs”的救援模式。不包含预构建的软件包。

作为如何使用 mini-memstick 镜像的示例，假设 USB 驱动器在你的机器上显示为 **/dev/da0**，类似以下命令应该可行：

```sh
# dd if=FreeBSD-10.2-RELEASE-amd64-mini-memstick.img of=/dev/da0 bs=1m conv=sync
```

请务必确保目标（of=）正确。

**FreeBSD/arm SD 卡镜像**：
可写入 SD 卡并用于引导受支持的 arm 系统。SD 卡镜像包含完整的 FreeBSD 安装，可安装到小至 512Mb 的 SD 卡上。

为方便没有控制台访问权限的用户，默认提供用户名为 `freebsd`、密码为 `freebsd` 的用户用于 **ssh(1)** 访问。此外，`root` 用户密码设为 `root`，强烈建议在获得系统访问权限后立即更改这两个用户的密码。

要将 FreeBSD/arm 镜像写入 SD 卡，请使用 **dd(1)** 工具，将 *KERNEL* 替换为系统对应的内核配置名称。

```sh
# dd if=FreeBSD-10.2-RELEASE-arm-armv6-KERNEL.img of=/dev/da0 bs=1m conv=sync
```

请务必确保目标（of=）正确。

## FTP

FreeBSD 10.2-RELEASE 可通过 ftp 从以下站点下载：

* <ftp://ftp.freebsd.org/pub/FreeBSD/releases/ISO-IMAGES/10.2/>

但在尝试此站点之前，请先检查你所在地区的镜像站点：

* `ftp://ftp.<your_country_code>.FreeBSD.org/pub/FreeBSD`

任何额外的镜像站点将标记为 ftp2、ftp3 等。

关于 FreeBSD 镜像站点的更多信息，请参阅：

* <https://docs.freebsd.org/en/books/handbook/mirrors/#ftp>

FreeBSD 10.2-RELEASE 虚拟机镜像可通过 ftp 从以下地址下载：

* <ftp://ftp.freebsd.org/pub/FreeBSD/releases/VM-IMAGES/10.2-RELEASE/>

关于安装 FreeBSD 或将现有机器更新至 10.2-RELEASE 的说明，请参阅：

* <https://www.FreeBSD.org/releases/10.2R/installation/>
