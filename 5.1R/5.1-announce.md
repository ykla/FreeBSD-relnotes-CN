# FreeBSD 5.1-RELEASE 发行公告

**日期：** 2003 年 6 月 9 日 09:36:57 -0700 (PDT)
**发件人：** Scott Long <scottl@FreeBSD.org>
**收件人：** <freebsd-announce@FreeBSD.org>
**主题：** [FreeBSD-Announce] ]FreeBSD 5.1 Released!

我很荣幸地宣布 FreeBSD 5.1-RELEASE 现已发布。此版本继续推进 FreeBSD 在先进多处理器和应用程序线程支持方面的进展，并包含许多改进和广受期待的功能：

- 实验性的 1:1 和 M:N 线程库为高效的应用程序多线程提供了内核支持。
- 对物理地址扩展（PAE）的支持使 Pentium Pro 及更高版本的 CPU 能访问多达 64GB 内存。
- 实验性的名称服务切换（NSS）基础设施允许企业无缝集成 LDAP 和 Active Directory 服务。
- 增强的“Jail”管理功能，允许一台服务器提供多个不同的“虚拟机”，同时减少管理员工作量。
- 新的设备驱动程序包括对 IBM/Adaptec ServeRAID 控制器的支持，扩展了对 USB 2.0 和 USB 以太网适配器的支持，以及对 Promise 串行 ATA 控制器的支持。
- 对 amd64 平台的实验性支持使 FreeBSD 能在单处理器 AMD Opteron 系统上运行。

尽管稳定性大幅提升且修复了许多 bug，FreeBSD 5.1 可能不适合所有用户。较为保守的用户可能更愿意继续使用 FreeBSD 4.X。有关各种权衡的信息，以及 FreeBSD 4.X 和 5.X 的未来计划说明，可在早期采用者指南中找到：

[http://www.FreeBSD.org/releases/5.1R/early-adopter.html](http://www.freebsd.org/releases/5.1R/early-adopter.html)

有关新功能和已知问题的完整列表，请参见发行说明和勘误列表：

[http://www.FreeBSD.org/releases/5.1R/errata.html](http://www.freebsd.org/releases/5.1R/errata.html)

## 可用性

FreeBSD 5.1-RELEASE 支持 i386、pc98、alpha、sparc64 和 ia64 架构，可使用引导软盘直接通过网络安装或复制到本地 NFS/FTP 服务器。所有架构的发行版现已提供。

请通过从我们的支持厂商处购买介质来继续支持 FreeBSD 项目。以下公司将提供基于 FreeBSD 5.1 的产品：
