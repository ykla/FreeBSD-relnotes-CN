# FreeBSD 14.0-RELEASE 发布计划

## 引言

这是 FreeBSD 14.0 的发布计划。
有关发行工程流程的更多信息，请参阅网站的 [发行工程](https://www.freebsd.org/releng/) 部分。

关于即将到来的发行和已知问题的一般性讨论，请发送至公开的 [freebsd-stable](mailto:FreeBSD-stable@FreeBSD.org) 邮件列表。

## 日程

在 main 分支冻结期间，发行工程团队可能会创建 ALPHA 快照以供测试之用。由于 ALPHA 快照的频率和总数取决于 main 分支冻结的时间长度，因此不提供官方时间表。

有关 ALPHA 快照可用性的公告将发送至 [freebsd-current](mailto:freebsd-current@FreeBSD.org) 和 [freebsd-snapshots](mailto:freebsd-snapshots@FreeBSD.org) 邮件列表。
