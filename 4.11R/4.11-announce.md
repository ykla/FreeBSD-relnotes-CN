# FreeBSD 4.11-RELEASE 发行公告

**日期：** 2005 年 1 月 25 日 12:02:10 -0500
**发件人：** Ken Smith <kensmith@FreeBSD.org>
**收件人：** <freebsd-announce@FreeBSD.org>
**主题：** FreeBSD 4.11-RELEASE is now available

发布工程团队很高兴地宣布 FreeBSD 4.11-RELEASE 现已发布，这是 FreeBSD 旧版开发分支的最新发行版本。自 2004 年 5 月的 FreeBSD 4.10-RELEASE 以来，我们对基本系统中的许多软件程序进行了保守更新，处理了已知的安全问题，并进行了许多 bug 修复。

有关新功能、已知问题和最新消息的完整列表，请参见发行说明和勘误列表，地址如下：

- [http://www.FreeBSD.org/releases/4.11R/relnotes.html](http://www.freebsd.org/releases/4.11R/relnotes.html)
- [http://www.FreeBSD.org/releases/4.11R/errata.html](http://www.freebsd.org/releases/4.11R/errata.html)

FreeBSD 4.11 将成为“勘误分支”。除安全修复外，其他经过充分测试的基本功能修复也将提交至 RELENG_4_11 分支。安全公告和勘误通知均在 <freebsd-announce@freebsd.org> 邮件列表上公布。

预计这是 RELENG_4 分支的最后一个发行版本。大多数开发者现在专注于 RELENG_5 分支或 HEAD 中的前沿开发。

有关 FreeBSD 发布工程活动的更多信息，请参见：

[http://www.FreeBSD.org/releng/](http://www.freebsd.org/releng/)

## 可用性

FreeBSD 4.11-RELEASE 支持 i386 和 alpha 架构，可使用可引导媒体直接通过网络安装或复制到本地 NFS/FTP 服务器。两种架构的发行版现已提供。

请继续从我们的支持厂商处购买介质以支持 FreeBSD 项目。以下公司将提供基于 FreeBSD 4.11 的产品：

| 公司 | 网址 |
| :--- | :--- |
| FreeBSD Mall, Inc. | [http://www.freebsdmall.com/](http://www.freebsdmall.com/) |
| Daemon News | [http://www.bsdmall.com/freebsd1.html](http://www.bsdmall.com/freebsd1.html) |

如果你无法负担介质形式的 FreeBSD、不耐烦等待，或仅想用于推广目的，那么请务必下载 ISO 镜像。我们无法保证所有镜像站点都提供较大的 ISO 镜像。截至本公告发布时，可从以下站点获取。发行版镜像的 MD5 校验和附在本消息末尾。
