# FreeBSD 3.3-RELEASE 发行公告

**日期：** Fri, 17 Sep 1999 05:06:44 -0700
**发件人：** “Jordan K. Hubbard” <jkh@freebsd.org>
**主题：** FreeBSD 3.3-RELEASE is now available.

是的，又到了那个时候！我非常高兴地宣布 FreeBSD 3.3-RELEASE 已发布，这是我们 3.x-STABLE 分支系列的最新发行版。作为 FreeBSD 3.2（1999 年 5 月发行）的后续版本，新增了大量驱动，修复了许多错误，并处理了若干重要的安全问题。更多信息请参见[发行说明](../notes/)。

FreeBSD 3.3-RELEASE 可从 [ftp.freebsd.org](ftp://ftp.freebsd.org/pub/FreeBSD) 以及全球各地的 [FTP 镜像站点](https://www.FreeBSD.org/handbook/mirrors.html) 获取。也可向 [The FreeBSD Mall](http://www.freebsdmall.com/) 订购 CD，不久后即可发货，为 4 CD 套装，内含 x86 架构的安装文件，以及大量对程序员和终端用户都颇具价值的其他资料。（Alpha 架构的 3.3-RELEASE 可从 FTP 站点获取。）

注意：此 CD 套装销售所得的全部利润均用于支持 FreeBSD 项目！我们在 3.3-RELEASE 中还尝试了一项新举措，将 Walnut Creek CDROM 官方发行版中的[第 1 张光盘](ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/i386/ISO-IMAGES/3.3-install.cd0)通过[匿名 FTP](ftp://ftp.freebsd.org/pub/FreeBSD/) 提供。这是其 4 CD 套装中最重要的一张光盘，允许用户从单个 ISO 9660 镜像安装基本系统及其所有最重要的附加组件。我们这样做是因为 ISO 镜像正迅速成为分发操作系统发行版的首选格式，如果提供标准的 NFS/FTP 网络安装方法已不再足够，我们当然不会故意让 FreeBSD 难以“试用”。我们无法保证所有镜像站点都会提供这个相当大的安装镜像（660MB），但至少可从以下地址获取：

**ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/i386/ISO-IMAGES/3.3-install.cd0**

以及更传统的 3.3-RELEASE 安装文件。如果你买不起 CD、等不及，或只是想用于推广目的，那么尽管下载 ISO；否则请继续通过从 [Walnut Creek CDROM](http://www.freebsdmall.com) 购买其官方 CD 发行版来支持 FreeBSD 项目。
