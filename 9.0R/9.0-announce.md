# FreeBSD 9.0-RELEASE 发行公告

日期：2012 年 1 月 12 日

FreeBSD 发行工程团队很高兴地宣布 FreeBSD 9.0-RELEASE 现已可用。这是 stable/9 分支的首个发行版本，提升了 stable/8 并引入许多新功能。部分亮点：

* 新增安装程序 [bsdinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=bsdinstall&sektion=8&format=html)，并作为本发行版 ISO 镜像所使用的安装程序
* 快速文件系统现在支持软更新日志
* ZFS 更新至版本 28
* 更新后的 ATA/SATA 驱动程序支持 AHCI，已迁移至更新的 CAM 框架
* 高可用存储（HAST）框架
* 内核支持 Capsicum 能力模式，这是一套用于沙箱支持的实验性特性
* 用户级 DTrace
* TCP/IP 网络栈现在支持可插拔拥塞控制框架，并提供五种拥塞控制算法实现
* NFS 子系统已更新，新实现除支持 NFSv3 和 NFSv2 外，还支持 NFSv4
* 高性能 SSH（HPN-SSH）
* 扁平设备树（FDT），简化了嵌入式平台的 FreeBSD 配置
* powerpc 架构现在支持 Sony Playstation 3
* 已导入 LLVM 编译器基础设施和 clang
* GNOME 版本 2.32.1，KDE 版本 4.7.3

关于新功能和已知问题的完整列表，请参阅在线发行说明和勘误列表，地址如下：

## 可用性

FreeBSD 9.0-RELEASE 现已可用于 amd64、i386、ia64、powerpc、powerpc64 和 sparc64 架构。

FreeBSD 9.0 可以从可引导 ISO 镜像或通过网络安装。某些架构还支持从 USB 闪存盘安装。所需文件可通过 FTP 或 BitTorrent 下载，如下节所述。虽然一些较小的 FTP 镜像可能不包含所有架构，但它们通常都会包含较常见的架构，如 amd64 和 i386。

> **注意**：
>
> 在 amd64 和 i386 架构的 DVD 镜像上传至 FTP 发行服务器后不久，发现了一个问题。这些镜像此后已被替换，我们已留出足够时间让更新的镜像分发到所有承载该发行版的 FTP 服务器。如果你在本公告发布之前下载了 amd64 或 i386 DVD 镜像，建议将你下载的镜像校验和与本发行公告中提供的校验和进行核对。被替换镜像的唯一问题是 [sysinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=sysinstall&sektion=8&format=html) 无法用于安装 DVD 上的预构建软件包。除此之外，更新的镜像没有任何不同。损坏的 DVD 镜像从未在 BitTorrent 上提供。

发行版 ISO 镜像的 MD5 和 SHA256 校验和附于本消息底部。

发行版中所提供镜像的用途如下：

*dvd1*：
包含安装 FreeBSD 基本系统、一组旨在让图形工作站快速运行起来的预构建软件包所需的全部内容。它还支持引导进入基于“livefs”的救援模式。如果你能刻录并使用 DVD 大小的介质，这应该就是你所需的全部。

*disc1*：
包含 FreeBSD 基本系统。它还支持引导进入基于“livefs”的救援模式。不包含预构建软件包。

*bootonly*：
支持使用 CDROM 驱动器引导机器，但不包含从 CD 本身安装 FreeBSD 的安装发行版组件。你需要从 CD 引导后执行基于网络的安装（例如从 FTP 服务器）。

*memstick*：
可写入 USB 闪存盘（U 盘），用于在能够从 USB 驱动器引导的机器上执行安装。它还支持引导进入基于“livefs”的救援模式。不包含预构建软件包。

作为使用 memstick 镜像的一个示例，假设 USB 驱动器在你的机器上显示为 **/dev/da0**，类似如下命令应该可行：

```sh
# dd if=FreeBSD-9.0-RELEASE-amd64-memstick.img of=/dev/da0 bs=10240 conv=sync
```

务必确保目标（of=）正确。
