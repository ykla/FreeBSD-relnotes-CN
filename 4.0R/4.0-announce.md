# FreeBSD 4.0-RELEASE 发行公告

**日期：** Tue, 14 Mar 2000 22:29:43 -0800 (PST)
**发件人：** “Jordan K. Hubbard” <jkh@zippy.cdrom.com>
**收件人：** <announce@FreeBSD.org>
**主题：** 4.0-RELEASE is now available

虽然稍晚了一些，但希望因此更加完善，它终于来了。我非常高兴地宣布 FreeBSD 4.0-RELEASE 发布。这是 4.x-stable（RELENG_4）分支的第一个发行版，相较于 FreeBSD 3.4 包含许多重大改进。新功能列表太长，此处不再逐一列举，详情请参见发行说明。

FreeBSD 4.0-RELEASE 可从 ftp.FreeBSD.org 及全球各地的 FTP 镜像站点获取。也可向 [The FreeBSD Mall](http://www.freebsdmall.com/) 订购 CD，不久后将以 4 CD 套装形式发货。4.0 将提供两套 CD：一套包含 x86 架构的安装文件（以及大量对程序员和终端用户都颇具价值的其他资料），另一套面向 DEC Alpha 架构机器。

与往常一样，Walnut Creek CDROM 官方发行版（两种架构）的第 1 张光盘在最终编译完成后，也将通过匿名 FTP 提供。请关注主 FTP 站点了解详情。我们无法保证所有镜像站点都会提供这个相当大的安装镜像（660MB），但至少可从以下地址获取：

**ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/i386/ISO-IMAGES/4.0-install.iso**
**ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/alpha/ISO-IMAGES/4.0-install.iso**

这些文件允许用户从单个可引导镜像安装基本系统及其所有最重要的附加组件，该镜像可由大多数 CD 创建软件作为原始 ISO 9660 镜像写入。

尽管我们免费提供安装 CD，但也希望你继续通过向 FreeBSD Mall 购买其官方 CD 发行版来支持 FreeBSD 项目。每笔销售的一部分用于支持 FreeBSD 的开发和整体基础设施，对此我们深表感谢。
