# FreeBSD 13.2-RELEASE 自述文件

## 摘要

本文简要介绍 FreeBSD 13.2-RELEASE。
包括如何获取 FreeBSD 的相关信息、联系 FreeBSD 项目的各种方式，以及其他信息来源的指引。

## 引言

本发行版是 FreeBSD 13.2-RELEASE，位于 13-STABLE 分支的最新节点。

## 关于 FreeBSD

FreeBSD 是一款源自 4.4 BSD Lite2 的开源操作系统，适用于 64 位 x86 PC 硬件（amd64）、32 位 x86 PC 硬件（i386）、64 位 Arm ARMv8-A AArch64 系统（aarch64）、32 位 Arm ARMv6/7 系统（armv6/7）、PowerPC 系统（powerpc、powerpc64）、RISC-V（riscv64）系统和 MIPS 系统（mips、mips64）。

FreeBSD 兼容各种外设和配置，可用于从软件开发、游戏到互联网服务提供的各类用途。

本 FreeBSD 发行版包含运行此类系统所需的全部内容，包括内核和基本系统中所有实用工具的完整源代码。
安装源代码发行版后，可通过一条命令从头重新编译整个系统，这使其成为学生、研究人员或单纯想了解系统工作原理的用户的理想选择。

本发行版还提供大量第三方移植软件（Ports 集合），便于获取和安装所有你喜爱的传统 UNIX 实用工具。
每个 Port 由一组脚本组成，用于获取、配置、构建和安装某个软件，只需一条命令即可完成。
大量 Port，从编辑器到编程语言再到图形应用程序，使 FreeBSD 成为功能强大且全面的操作环境，远超许多商业版 UNIX 所提供的功能。
大多数 Port 还提供预编译的软件包，可通过 [pkg(8)](https://man.freebsd.org/cgi/man.cgi?query=pkg&sektion=8&format=html) 工具快速安装。

## 目标受众

本 FreeBSD 发行版适合所有用户。
它经过了一段时间的测试和质量保证，以确保最高的可靠性和稳定性。

## 获取 FreeBSD

FreeBSD 可通过多种方式获取。
最便捷的方式是下载发行版镜像。
最近发行版的镜像链接可在 FreeBSD 网站 [下载页面](https://www.freebsd.org/where/) 找到。
该页面包含关于不同类型镜像的信息，以及获取 FreeBSD 副本的其他方式。

## 联系 FreeBSD 项目

### 电子邮件和邮件列表

如有一般问题或技术支持，请联系 [freebsd-questions 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-questions)。

如果你跟踪 `13-STABLE` 分支，必须订阅 [freebsd-stable 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-stable)，以了解可能影响系统使用和维护的开发动态和变更。

作为主要由志愿者推动的项目，FreeBSD 项目始终欢迎愿意提供帮助的人手——期望实现的增强功能远多于用于实现这些功能的时间。
如需就技术问题联系开发者或提供帮助，请联系 [freebsd-hackers 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-hackers)。

上述邮件列表的流量可能较大。
如果你的邮件访问速度较慢或费用较高，或只关注重大公告，可订阅 [freebsd-announce 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-announce)。

所有公共邮件列表均可供任何希望加入的人订阅。
超过一百个列表是公开的；<https://lists.freebsd.org/> 列出了全部列表，并提供浏览归档、订阅和退订的相关信息。

### 提交问题报告

欢迎提出建议、提交缺陷报告和贡献代码。
如发现任何问题，请不要犹豫，及时报告。
附带修复的缺陷报告当然更受欢迎。

从具有 Internet 连接的机器提交缺陷报告的首选方式是使用 [Bugzilla 缺陷跟踪器](https://bugs.freebsd.org)。
以此方式提交的问题报告（PR）将被归档并跟踪其处理进度；FreeBSD 开发者将尽最大努力尽快响应所有报告的缺陷。
FreeBSD 网站上提供 [所有活跃 PR 的列表](https://bugs.freebsd.org/search/)；该列表有助于了解其他用户遇到的潜在问题。

更多信息，请参阅 FreeBSD 网站上的 [编写 FreeBSD 问题报告](https://docs.freebsd.org/en/articles/problem-reports/)，其中包含编写和提交有效问题报告的诸多实用提示。

## 延伸阅读

关于 FreeBSD 的信息来源很多；部分包含在本发行版中，其他则可在线获取或以印刷版本形式获取。

### 发行版文档

若干其他文件提供了关于本发行版的更具体信息。
这些文件以多种格式提供。
大多数发行版将同时包含 ASCII 文本（`.TXT`）和 HTML（`.HTM`）版本。
部分发行版还可能包含其他格式，如便携文档格式（`.PDF`）。

* `README.TXT`：本文件，提供关于 FreeBSD 的一般信息以及获取发行版的简要说明。
* `RELNOTES.TXT`：发行说明，展示 FreeBSD 13.2-RELEASE 相对于前一版本（FreeBSD 13.1-RELEASE）的新增和不同之处。
* `HARDWARE.TXT`：硬件兼容列表，展示已测试并已知可与 FreeBSD 配合使用的设备。
* `ERRATA.TXT`：发行版勘误。本文件包含发布后期及发布后的最新信息，主要适用于发行版（而非快照）。在安装 FreeBSD 发行版之前，务必查阅本文件，因为它包含自发行版创建以来发现并修复的问题的最新信息。

安装过程中，可通过文档菜单访问这些文档。
系统安装完成后，可重新运行 [bsdinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=bsdinstall&sektion=8&format=html) 工具再次访问此菜单。

> **注意**：
>
> 要了解任何最新消息或发布后的问题，在安装任何发行版之前，阅读其勘误文件极为重要。
> 随各发行版附带的勘误文件（很可能就在本文件旁边）按定义已经过时，但其他副本在互联网上保持更新，应作为本发行版的当前勘误查阅。
> 这些其他勘误副本位于 <https://www.FreeBSD.org/releases>（以及任何保持该位置最新镜像的站点）。

### 手册页

与几乎所有 UNIX 类操作系统一样，FreeBSD 包含一套手册页，可 [在线](https://man.freebsd.org) 访问或通过 [man(1)](https://man.freebsd.org/cgi/man.cgi?query=man&sektion=1&format=html) 命令访问。

通常，这些页面提供关于 FreeBSD 用户可用的不同命令和 API 的信息。

部分页面旨在提供特定主题的信息。
值得注意的例子包括 [tuning(7)](https://man.freebsd.org/cgi/man.cgi?query=tuning&sektion=7&format=html)（性能调优指南）、[security(7)](https://man.freebsd.org/cgi/man.cgi?query=security&sektion=7&format=html)（FreeBSD 安全简介）和 [style(9)](https://man.freebsd.org/cgi/man.cgi?query=style&sektion=9&format=html)（内核编码风格指南）。

### 书籍和文章

FreeBSD 项目维护的文档包括非常实用的 *FreeBSD 手册*（包含安装和升级操作系统的详细指南）和 *FreeBSD FAQ*（常见问题解答）。
[手册](https://docs.freebsd.org/en/books/handbook/) 和 [FAQ](https://docs.freebsd.org/en/books/faq/) 在 [文档门户](https://docs.freebsd.org/) 和镜像站点在线提供。

其他项目维护的书籍和文章更加专业化——涵盖广泛的主题，从邮件列表的有效使用，到 FreeBSD 与其他操作系统的双启动，再到新提交者的指南。
与手册和 FAQ 一样，这些文档位于门户、镜像和 `doc` 仓库中。

如需 HTML 和 PDF 格式的离线文档：可安装特定语言的软件包，如 [misc/freebsd-doc-en](https://cgit.freebsd.org/ports/tree/misc/freebsd-doc-en/)（`-en` 表示英语），或多语言软件包 [misc/freebsd-doc-all](https://cgit.freebsd.org/ports/tree/misc/freebsd-doc-all/)。
或者，使用 `doc` 仓库的副本从源代码构建和安装。

关于 FreeBSD 的其他书籍和文档列表可在 FreeBSD 手册的 [参考文献](https://docs.freebsd.org/en/books/handbook/bibliography/) 中找到。
由于 FreeBSD 深厚的 UNIX 血统，许多为 UNIX 系统编写的其他文章和书籍同样适用，其中部分也列入参考文献中。
