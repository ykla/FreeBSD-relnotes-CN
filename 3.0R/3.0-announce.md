# FreeBSD 3.0-RELEASE 发行公告

**日期：** Fri, 16 Oct 1998 13:19:03 -0700
**发件人：** “Jordan K. Hubbard” <jkh@time.cdrom.com>
**主题：** FreeBSD 3.0 is now released!

在 FreeBSD 项目志愿者们付出堪称英勇的努力之后，期待已久的 FreeBSD 3.0 终于发布了！

本发行版主要面向开发者和早期采用者，不过也有许多 ISP 报告在生产环境中使用它取得了良好的效果（当然，我们只建议技术最为娴熟的用户这么做）。更多信息请参见[发行说明](notes.html)。

FreeBSD 3.0-RELEASE 可从 [ftp.FreeBSD.org](ftp://ftp.FreeBSD.org/pub/FreeBSD) 以及全球各地的 [FTP 镜像站点]({handbook}#mirrors) 获取。也可向 [Walnut Creek CDROM](http://www.cdrom.com/) 订购 CD，不久后即可发货，为 4 CD 套装，内含大量对程序员和普通用户都颇具价值的额外资料。

**重要提示：** 此 CD 套装销售所得的全部利润均用于支持 FreeBSD 项目！

正如高科技领域的许多企业一样，Walnut Creek CDROM 意识到，要使面向新兴市场的任何产品获得增长，就必须为这种增长投入大量资源，即便这意味着放弃短期利润。Walnut Creek CDROM 是目前唯一采取此种做法的 CDROM 供应商，我真诚地希望你能通过购买（或让别人购买 :) 他们的某张 CD 来帮助支持本项目。谢谢！

在尝试访问中央 FTP 站点之前，请先前往以下地址查看你所在区域的镜像站点：

**ftp://ftp.<yourdomain>.FreeBSD.org/pub/FreeBSD**

任何额外的镜像站点将依次命名为 ftp2、ftp3 等。
