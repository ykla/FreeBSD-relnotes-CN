## 1 引言

本发行版是 FreeBSD 7.1-RELEASE，即 7.1-STABLE 分支上的最新版本。

### 1.1 关于 FreeBSD

FreeBSD 是基于 4.4 BSD Lite 的操作系统，适用于 AMD64 和 Intel EM64T 架构 PC 硬件（amd64）、Intel、AMD、Cyrix 或 NexGen“x86”架构 PC 硬件（i386）、Intel Itanium 处理器计算机（ia64）、NEC PC-9801/9821 系列 PC 及兼容机（pc98），以及 UltraSPARC 机器（sparc64）。适用于 PowerPC（powerpc）和 MIPS（mips）架构的版本目前也在开发中。FreeBSD 兼容各种外设和配置，可用于从软件开发到游戏，再到互联网服务提供的各种用途。

此 FreeBSD 发行版包含运行此类系统所需的所有内容，包括内核和基本发行版中所有工具的完整源代码。安装源代码发行版后，你只需一条命令即可从零开始重新编译整个系统，使其非常适合学生、研究人员或仅想了解其工作原理的用户。

还提供了大量第三方移植软件集合（“Ports”），便于你为 FreeBSD 获取和安装所有喜爱的传统 UNIX 工具。每个“port”由一组脚本组成，可通过一条命令检索、配置、编译和安装软件。超过 19,000 个 Ports，从编辑器到编程语言再到图形应用程序，使 FreeBSD 成为强大而全面的操作系统环境，远超许多商业 UNIX 版本所提供的功能。大多数 Port 还提供预编译的“包”，可从安装程序快速安装。

### 1.2 目标受众

此 FreeBSD 发行版适合所有用户。它经过了一段时间的测试和质量保证检查，以确保最高的可靠性和可依赖性。

## 2 获取 FreeBSD

可以通过多种方式获取 FreeBSD。本节主要介绍那些用于获取完整 FreeBSD 发行版的方式，而非更新现有安装。

### 2.1 CDROM 和 DVD

可从多家出版商订购 CDROM 或 DVD 形式的 FreeBSD -RELEASE 发行版。这通常是新安装获取 FreeBSD 的最便捷方式，因为它提供了一种在必要时快速重新安装系统的便捷方法。某些发行版包含 FreeBSD Ports 中的一些可选预编译“包”，或其他附加材料。

项目已知的 CDROM 和 DVD 出版商列表列于手册的 [“获取 FreeBSD”附录](https://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/mirrors.html) 中。

### 2.2 FTP

你可以使用 FTP 从 [ftp://ftp.FreeBSD.org/](ftp://ftp.freebsd.org/)（官方 FreeBSD 发行站点）或其任何“镜像”站点检索 FreeBSD 及其任何可选包。

镜像 FreeBSD 的位置列表可在手册的 [FTP 站点](https://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/mirrors-ftp.html) 部分或 [http://mirrorlist.FreeBSD.org/](http://mirrorlist.freebsd.org/) 网页上找到。强烈建议从网络距离较近的镜像下载发行版。

欢迎额外的镜像站点。联系 [freebsd-admin@FreeBSD.org](mailto:freebsd-admin@FreeBSD.org) 了解成为官方镜像站点的更多详情。你还可以在 [镜像 FreeBSD](https://www.freebsd.org/doc/en_US.ISO8859-1/articles/hubs/) 文章中找到镜像站点的有用信息。

镜像通常包含用于创建 FreeBSD 发行版 CDROM 的 ISO 镜像。它们通常还包含软盘镜像（适用于适用的平台），以及通过网络进行安装所需的文件。最后，镜像站点通常包含最新发行版的一组软件包。

## 3 联系 FreeBSD 项目

### 3.1 电子邮件和邮件列表

如有任何问题或一般技术支持问题，请发送邮件至 [FreeBSD 一般问题邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-questions)。

如果你正在跟踪 7.1-STABLE 的开发工作，你**必须**加入 [FreeBSD-CURRENT 邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-current)，以了解可能影响你使用和维护系统的最新开发和更改。

由于 FreeBSD 项目很大程度上由志愿者推动，因此始终欢迎愿意提供帮助的额外人手——期望的增强功能数量远超时间所能实现的。要就技术事项联系开发者或提供帮助，请发送邮件至 [FreeBSD 技术讨论邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-hackers)。

请注意，这些邮件列表可能会有**大量**的流量。如果你的邮件访问速度慢或费用高昂，或仅希望了解主要的 FreeBSD 事件，你可能更适合订阅 [FreeBSD 公告邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-announce)。

所有邮件列表均欢迎任何愿意加入的人自由订阅。访问 [FreeBSD Mailman 信息页](http://www.freebsd.org/mailman/listinfo)。这将为你提供有关加入各种列表、访问归档等的更多信息。还有许多针对此处未提及的特殊兴趣小组的邮件列表；可从 Mailman 页面或 FreeBSD 网站的 [邮件列表部分](https://www.freebsd.org/support.html#mailing-list) 获取更多信息。

> **重要**
>
> 请**不要**向列表发送要求订阅的邮件。请使用 Mailman 界面。

### 3.2 提交问题报告

建议、错误报告和代码贡献始终受到重视——请随时报告你发现的任何问题。附带修复的错误报告当然更受欢迎。

从具有互联网邮件连接的机器提交错误报告的首选方法是使用 [send-pr(1)](http://www.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+7.1-RELEASE) 命令。以这种方式提交的“问题报告”（PR）将被归档并跟踪其进度；FreeBSD 开发者将尽力尽快响应所有报告的错误。FreeBSD 网站上提供 [所有活动 PR 的列表](http://www.freebsd.org/cgi/query-pr-summary.cgi)；此列表有助于了解其他用户遇到的潜在问题。

请注意，[send-pr(1)](http://www.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+7.1-RELEASE) 本身是一个 shell 脚本，应易于移植到非 FreeBSD 系统上。强烈建议使用此接口。如果由于某种原因无法使用 [send-pr(1)](http://www.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+7.1-RELEASE) 提交错误报告，你可以尝试将其发送至 [FreeBSD 问题报告邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-bugs)。

如需更多信息，FreeBSD 网站上的 [“撰写 FreeBSD 问题报告”](https://www.freebsd.org/doc/en_US.ISO8859-1/articles/problem-reports/) 提供了许多关于撰写和提交有效问题报告的有用提示。

## 4 延伸阅读

关于 FreeBSD 的信息来源很多；一些包含在此发行版中，另一些可在线或在印刷版本中获取。

### 4.1 发行文档

许多其他文件提供了关于此发行版的更具体信息。这些文件以多种格式提供。大多数发行版将包含 ASCII 文本（**.TXT**）和 HTML（**.HTM**）格式。某些发行版还可能包含其他格式，如便携文档格式（**.PDF**）。

- **README.TXT**：本文件，提供有关 FreeBSD 的一些一般信息以及获取发行版的简要说明。
- **RELNOTES.TXT**：发行说明，显示 FreeBSD 7.1-RELEASE 相对于上一版本（FreeBSD 7.0-RELEASE）的新增和不同之处。
- **HARDWARE.TXT**：硬件兼容列表，显示已测试并已知可与 FreeBSD 配合工作的设备。
- **ERRATA.TXT**：发行勘误。此文件包含最新的、发布后的信息，主要适用于发行版（而非快照）。在安装 FreeBSD 发行版之前查阅此文件非常重要，因为它包含自发行版创建以来已发现并修复的问题的最新信息。

在支持 [sysinstall(8)](http://www.freebsd.org/cgi/man.cgi?query=sysinstall&sektion=8&manpath=FreeBSD+7.1-RELEASE) 的平台上（目前为 amd64、i386、ia64、pc98 和 sparc64），这些文档通常可在安装期间通过文档菜单获得。系统安装后，可以通过重新运行 [sysinstall(8)](http://www.freebsd.org/cgi/man.cgi?query=sysinstall&sektion=8&manpath=FreeBSD+7.1-RELEASE) 工具重新访问此菜单。

> **注意**
>
> 在安装任何给定发行版之前，阅读其勘误极其重要，以了解任何“最新消息”或发布后的问题。随每个发行版附带的勘误文件（很可能就在本文件旁边）按定义过时，但其他副本会在互联网上保持更新，应作为此版本的“当前勘误”查阅。这些其他勘误副本位于 [http://www.FreeBSD.org/releases/](http://www.freebsd.org/releases/)（以及任何保持此位置最新镜像的站点）。

### 4.2 手册页

与几乎所有类 UNIX 操作系统一样，FreeBSD 附带一组联机手册页，可通过 [man(1)](http://www.freebsd.org/cgi/man.cgi?query=man&sektion=1&manpath=FreeBSD+7.1-RELEASE) 命令或 FreeBSD 网站上的 [超文本文档手册页网关](http://www.freebsd.org/cgi/man.cgi) 访问。通常，手册页提供有关 FreeBSD 用户可用的不同命令和 API 的信息。

在某些情况下，手册页用于提供特定主题的信息。此类手册页的著名示例包括 [tuning(7)](http://www.freebsd.org/cgi/man.cgi?query=tuning&sektion=7&manpath=FreeBSD+7.1-RELEASE)（性能调优指南）、[security(7)](http://www.freebsd.org/cgi/man.cgi?query=security&sektion=7&manpath=FreeBSD+7.1-RELEASE)（FreeBSD 安全简介）和 [style(9)](http://www.freebsd.org/cgi/man.cgi?query=style&sektion=9&manpath=FreeBSD+7.1-RELEASE)（内核编码风格指南）。

### 4.3 书籍和文章

FreeBSD 项目维护了两个非常有用的 FreeBSD 相关信息集合：FreeBSD 手册和 FreeBSD FAQ（常见问题解答）。[手册](https://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/) 和 [FAQ](https://www.freebsd.org/doc/en_US.ISO8859-1/books/faq/) 的在线版本始终可从 [FreeBSD 文档页面](http://www.freebsd.org/docs.html)或其镜像获取。如果安装 **doc** 发行集，你可以使用 Web 浏览器在本地阅读手册和 FAQ。特别要注意，手册包含逐步安装 FreeBSD 的指南。

FreeBSD 项目还维护了许多涵盖更专业的 FreeBSD 相关主题的在线书籍和文章。这些材料涵盖广泛的主题，从有效使用邮件列表，到 FreeBSD 与其他操作系统的双重引导，再到新提交者的指南。与手册和 FAQ 一样，这些文档可从 FreeBSD 文档页面或 **doc** 发行集中获取。

其他关于 FreeBSD 的书籍和文档列表可在 FreeBSD 手册的 [参考文献](https://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/bibliography.html) 中找到。由于 FreeBSD 的深厚 UNIX 传统，许多为 UNIX 系统撰写的其他文章和书籍同样适用，其中一些也列在参考文献中。
