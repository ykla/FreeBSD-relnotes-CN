## 1 引言

本发行版是 FreeBSD 4.8-RELEASE，即 4-STABLE 分支上的最新版本。

### 1.1 关于 FreeBSD

FreeBSD 是一个基于 4.4 BSD Lite 的操作系统，适用于 Intel、AMD、Cyrix 或 NexGen“x86”架构 PC 硬件以及 Compaq（前身为 DEC）Alpha 计算机。适用于 IA64、PowerPC 和 Sparc64 架构的版本也在开发中。FreeBSD 兼容各种外设和配置，可用于从软件开发到游戏，再到互联网服务提供的各种用途。

此 FreeBSD 发行版包含运行此类系统所需的所有内容，包括内核和基本发行版中所有工具的完整源代码。安装源代码发行版后，你只需一条命令即可从零开始重新编译整个系统，使其非常适合学生、研究人员或仅想了解其工作原理的用户。

还提供了大型第三方移植软件集合（“Ports 集合”），使你能轻松获取并安装所有喜爱的传统 UNIX 工具到 FreeBSD。每个“port”由一组脚本组成，可通过一条命令检索、配置、编译和安装软件。超过 8,300 个 Ports，从编辑器到编程语言再到图形应用程序，使 FreeBSD 成为一个强大而全面的操作系统环境，远超许多商业 UNIX 版本所提供的功能。大多数 Ports 还提供预编译的“软件包”，可从安装程序快速安装。

### 1.2 目标受众

此 FreeBSD 发行版适合所有用户。它经过了一段时间的测试和质量保证检查，以确保最高的可靠性和可信度。

## 2 获取 FreeBSD

可以通过多种方式获取 FreeBSD。本节主要介绍那些用于获取完整 FreeBSD 发行版的方式，而非更新现有安装。

### 2.1 CDROM 和 DVD

可从多家出版商订购 CDROM 或 DVD 形式的 FreeBSD -RELEASE 发行版。这通常是新安装获取 FreeBSD 的最便捷方式，因为它提供了一种在必要时快速重新安装系统的便捷方法。某些发行版包含 FreeBSD Ports 集合中的一些可选预编译“软件包”。

项目已知的 CDROM 和 DVD 出版商列表列于手册的[“获取 FreeBSD”附录](http://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/mirrors.html)中。

### 2.2 FTP

你可以使用 FTP 从 [ftp://ftp.FreeBSD.org/](ftp://ftp.freebsd.org/)（官方 FreeBSD 发行站点）或其任意“镜像”检索 FreeBSD 及其任意可选软件包。

镜像 FreeBSD 的站点列表可在手册的 [FTP 站点](http://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/mirrors-ftp.html)部分或 [http://mirrorlist.FreeBSD.org/](http://mirrorlist.freebsd.org/) 网页上找到。强烈建议从网络距离较近的镜像下载发行版。

欢迎提供额外的镜像站点。有关成为官方镜像站点的更多详细信息，请联系 <[freebsd-admin@FreeBSD.org](mailto:freebsd-admin@FreeBSD.org)>。

镜像通常包含开始安装所需的软盘镜像，以及安装过程本身所需的发行文件。许多镜像还包含创建 FreeBSD 发行版 CDROM 所需的 ISO 镜像。

## 3 联系 FreeBSD 项目

### 3.1 电子邮件和邮件列表

如有任何问题或一般技术支持问题，请发送邮件至 FreeBSD 一般问题邮件列表 <[freebsd-questions@FreeBSD.org](mailto:freebsd-questions@FreeBSD.org)>。

如果你正在跟踪 -STABLE 开发工作，你**必须**加入 FreeBSD-STABLE 邮件列表 <[freebsd-stable@FreeBSD.org](mailto:freebsd-stable@FreeBSD.org)>，以便及时了解可能影响你使用和维护系统的最新开发和变更。

FreeBSD 项目在很大程度上是一个志愿者项目，始终欢迎有意愿提供帮助的人手——期望的增强功能已远远超过有时间实现的数量。要就技术事项联系开发者或提供帮助，请发送邮件至 FreeBSD 技术讨论邮件列表 <[freebsd-hackers@FreeBSD.org](mailto:freebsd-hackers@FreeBSD.org)>。

请注意，这些邮件列表可能经历**大量**流量。如果你的邮件访问缓慢或昂贵，或仅想了解 FreeBSD 的重大事件，你可能更愿意订阅 FreeBSD 公告邮件列表 <[freebsd-announce@FreeBSD.org](mailto:freebsd-announce@FreeBSD.org)>。

所有邮件列表均可由任何愿意加入的人自由加入。请访问 [FreeBSD Mailman 信息页面](http://lists.freebsd.org/mailman/listinfo)。这将为你提供有关加入各种列表、访问归档等的更多信息。还有许多针对此处未提及的特殊兴趣群体的邮件列表；可通过 majordomo 或 FreeBSD 网站的[邮件列表部分](http://www.freebsd.org/support.html#mailing-list)获取更多信息。

> **重要：** **不要**向列表发送请求订阅的邮件。请改用 <[majordomo@FreeBSD.org](mailto:majordomo@FreeBSD.org)> 地址。

### 3.2 提交问题报告

建议、bug 报告和代码贡献始终受到重视——请不要犹豫报告你发现的任何问题。附带修复的 bug 报告当然更受欢迎。

从具有互联网邮件连接的机器提交 bug 报告的首选方法是使用 [send-pr(1)](https://man.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+4.8-RELEASE) 命令或使用 [http://www.FreeBSD.org/send-pr.html](http://www.freebsd.org/send-pr.html) 上的 Web 表单。以这种方式提交的“问题报告”（PR）将被归档并跟踪其进度；FreeBSD 开发者将尽最大努力尽快响应所有报告的 bug。FreeBSD 网站上提供[所有活动 PR 的列表](http://www.freebsd.org/cgi/query-pr-summary.cgi)；此列表可用于了解其他用户遇到的潜在问题。

请注意，[send-pr(1)](https://man.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+4.8-RELEASE) 本身是一个 shell 脚本，即使移至非 FreeBSD 系统也应易于使用。强烈推荐使用此接口。如果由于某种原因你无法使用 [send-pr(1)](https://man.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+4.8-RELEASE) 提交 bug 报告，可以尝试将其发送至 FreeBSD 问题报告邮件列表 <[freebsd-bugs@FreeBSD.org](mailto:freebsd-bugs@FreeBSD.org)>。

更多信息，请参见 FreeBSD 网站上的[“编写 FreeBSD 问题报告”](http://www.freebsd.org/doc/en_US.ISO8859-1/articles/problem-reports/)，其中包含许多有关编写和提交有效问题报告的有用提示。

## 4 进一步阅读

有关 FreeBSD 的信息来源很多；一些包含在本发行版中，而另一些则可在线或以印刷版本获取。

### 4.1 发行文档

若干其他文件提供了有关此发行版的更具体信息。这些文件以多种格式提供。大多数发行版将包含 ASCII 文本（`.TXT`）和 HTML（`.HTM`）两种版本。某些发行版还可能包含其他格式，如 PostScript（`.PS`）或便携文档格式（`.PDF`）。

- `README.TXT`：此文件提供有关 FreeBSD 的一些一般信息以及获取发行版的简要说明。
- `RELNOTES.TXT`：发行说明，展示 FreeBSD 4.8-RELEASE 相对于前一版本（FreeBSD 4.7-RELEASE）的新内容和不同之处。
- `HARDWARE.TXT`：硬件兼容列表，展示已测试并已知可与 FreeBSD 配合使用的设备。
- `INSTALL.TXT`：从发行介质安装 FreeBSD 的安装说明。
- `ERRATA.TXT`：发行勘误。可在此文件中找到最新发布的发行后信息，主要适用于发行版（而非快照）。在安装 FreeBSD 发行版之前，请务必查阅此文件，因为它包含自发行版创建以来发现并修复的问题的最新信息。

> **注意：** 其中若干文档（特别是 `RELNOTES.TXT`、`HARDWARE.TXT` 和 `INSTALL.TXT`）包含特定于某一硬件架构的信息。例如，alpha 发行说明包含不适用于 i386 的信息，反之亦然。每个文档适用的架构将在该文档的标题中列出。

这些文档通常可在安装期间的文档菜单中找到。系统安装后，你可以通过运行 [sysinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=sysinstall&sektion=8&manpath=FreeBSD+4.8-RELEASE) 工具重新访问此菜单。

> **注意：** 在安装任何给定发行版之前，请务必阅读其勘误，以了解任何“最新消息”或发行后问题。随每个发行版附带的勘误文件（很可能就在此文件旁边）按定义已过时，但其他副本会在互联网上保持更新，应作为此版本的“当前勘误”查阅。这些其他勘误副本位于 [http://www.FreeBSD.org/releases/](http://www.freebsd.org/releases/)（以及任何保持此位置最新镜像的站点）。

### 4.2 手册页

与几乎所有类 UNIX 操作系统一样，FreeBSD 附带一组联机手册页，可通过 [man(1)](https://man.freebsd.org/cgi/man.cgi?query=man&sektion=1&manpath=FreeBSD+4.8-RELEASE) 命令或 FreeBSD 网站上的[超文本手册页网关](https://man.freebsd.org/cgi/man.cgi)访问。通常，手册页提供有关 FreeBSD 用户可用的不同命令和 API 的信息。

在某些情况下，手册页是为特定主题编写的。此类手册页的著名示例包括 [tuning(7)](https://man.freebsd.org/cgi/man.cgi?query=tuning&sektion=7&manpath=FreeBSD+4.8-RELEASE)（性能调优指南）、[security(7)](https://man.freebsd.org/cgi/man.cgi?query=security&sektion=7&manpath=FreeBSD+4.8-RELEASE)（FreeBSD 安全简介）和 [style(9)](https://man.freebsd.org/cgi/man.cgi?query=style&sektion=9&manpath=FreeBSD+4.8-RELEASE)（内核编码风格指南）。

### 4.3 书籍和文章

由 FreeBSD 项目维护的两个非常有用的 FreeBSD 相关信息集合是 FreeBSD 手册和 FreeBSD FAQ（常见问题文档）。[手册](http://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/)和 [FAQ](http://www.freebsd.org/doc/en_US.ISO8859-1/books/faq/) 的在线版本始终可从 [FreeBSD 文档页面](http://www.freebsd.org/docs.html)或其镜像获取。如果你安装 `doc` 发行集，可以使用 Web 浏览器在本地阅读手册和 FAQ。

由 FreeBSD 项目维护的若干在线书籍和文章涵盖了更专业的 FreeBSD 相关主题。这些材料涵盖广泛的主题，从邮件列表的有效使用，到 FreeBSD 与其他操作系统的双启动，再到新提交者的指南。与手册和 FAQ 一样，这些文档可从 FreeBSD 文档页面或 `doc` 发行集中获取。

有关 FreeBSD 的其他书籍和文档列表可在 FreeBSD 手册的[参考文献](http://www.freebsd.org/doc/en_US.ISO8859-1/books/handbook/bibliography.html)中找到。由于 FreeBSD 具有深厚的 UNIX 血统，许多为 UNIX 系统编写的其他文章和书籍同样适用，其中一些也列于参考文献中。
