## 摘要

本文简要介绍 FreeBSD 8.1-RELEASE。包括如何获取 FreeBSD 的相关信息、联系 FreeBSD 项目的各种方式，以及其他信息来源的指引。

## 1 引言

本发行版是 FreeBSD 8.1-RELEASE，位于 8-STABLE 分支的最新节点。

### 1.1 关于 FreeBSD

FreeBSD 是一款基于 4.4 BSD Lite 的操作系统，适用于 AMD64 和 Intel EM64T 架构的 PC 硬件（amd64）、Intel、AMD、Cyrix 或 NexGen“x86”架构的 PC 硬件（i386）、Intel Itanium 处理器架构的计算机（ia64）、NEC PC-9801/9821 系列个人电脑及兼容机（pc98）以及 UltraSPARC® 机器（sparc64）。ARM®（arm）、MIPS®（mips）和 PowerPC®（powerpc）架构的版本目前也在开发中。FreeBSD 兼容各种外设和配置，可用于从软件开发、游戏到互联网服务提供的各类用途。

本 FreeBSD 发行版包含运行此类系统所需的全部内容，包括内核和基本系统中所有实用工具的完整源代码。安装源代码发行版后，可通过一条命令从头重新编译整个系统，这使其成为学生、研究人员或单纯想了解系统工作原理的用户的理想选择。

本项目还提供大量第三方移植软件集合（“Ports”），便于获取和安装所有你喜爱的传统 UNIX® 实用工具。每个 Port 由一组脚本组成，用于检索、配置、构建和安装某个软件，只需一条命令即可完成。超过 20,000 个 Port，从编辑器到编程语言再到图形应用程序，使 FreeBSD 成为功能强大且全面的操作环境，远超许多商业版 UNIX 所提供的功能。大多数 Port 还提供预编译的软件包，可通过安装程序快速安装。

### 1.2 目标受众

本 FreeBSD 发行版适合所有用户。它经过了一段时间的测试和质量保证检查，以确保最高的可靠性和稳定性。

## 2 获取 FreeBSD

FreeBSD 可通过多种方式获取。本节重点介绍主要用于获取完整 FreeBSD 发行版的方式，而非更新现有安装。

### 2.1 CDROM 和 DVD

FreeBSD -RELEASE 发行版可从多家出版商订购 CDROM 或 DVD。这通常是新安装时获取 FreeBSD 最便捷的方式，因为它提供了一种在必要时快速重装系统的便利方法。部分发行版包含 FreeBSD Ports 中的一些可选预编译软件包，或其他附加材料。

项目已知的 CDROM 和 DVD 出版商列表见手册的 [“获取 FreeBSD”](http://www.FreeBSD.org/doc/en_US.ISO8859-1/books/handbook/mirrors.html) 附录。

### 2.2 FTP

你可以使用 FTP 从 <ftp://ftp.FreeBSD.org/>（FreeBSD 官方发布站点）或其任一“镜像”站点检索 FreeBSD 及其任一或全部可选软件包。

镜像 FreeBSD 的站点位置列表见手册的 [FTP 站点](http://www.FreeBSD.org/doc/en_US.ISO8859-1/books/handbook/mirrors-ftp.html) 部分，或 <http://mirrorlist.FreeBSD.org/> 网页。强烈建议从网络距离较近的镜像站点下载发行版。

欢迎增加额外的镜像站点。请联系 <freebsd-admin@FreeBSD.org> 了解成为官方镜像站的详细信息。你也可以在 [镜像 FreeBSD](http://www.FreeBSD.org/doc/en_US.ISO8859-1/articles/hubs/) 文章中找到对镜像站点有用的信息。

镜像站点通常包含用于创建 FreeBSD 发行版 CDROM 的 ISO 镜像。它们通常还包含软盘镜像（适用于支持的平台），以及通过网络安装所需的文件。最后，镜像站点通常包含最新发行版的软件包集合。

## 3 联系 FreeBSD 项目

### 3.1 电子邮件和邮件列表

如有任何问题或一般技术支持问题，请发送邮件至 [FreeBSD 一般问题邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-questions)。

如果你在跟踪 8-STABLE 的开发工作，你 **必须** 加入 [FreeBSD-CURRENT 邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-current)，以便了解可能影响你使用和维护系统的最新开发和变更。

作为一项主要依靠志愿者的工作，FreeBSD 项目始终欢迎愿意提供帮助的人手——期望的增强功能数量已远超可实现的时间。如需就技术问题联系开发者或提供帮助，请发送邮件至 [FreeBSD 技术讨论邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-hackers)。

请注意，这些邮件列表可能会有 **大量** 邮件。如果你的邮件访问速度较慢或费用较高，或仅希望了解 FreeBSD 的重大事件，你可能更愿意订阅 [FreeBSD 公告邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-announce)。

所有邮件列表均欢迎任何愿意加入的人士。请访问 [FreeBSD Mailman 信息页面](http://www.FreeBSD.org/mailman/listinfo)。这将提供关于加入各种列表、访问归档等更多信息。还有许多针对特定兴趣小组的邮件列表未在此处提及；更多信息可从 Mailman 页面或 FreeBSD 网站的 [邮件列表部分](http://www.FreeBSD.org/support.html#mailing-list) 获取。

>**重要**
>
> 请 **不要** 向邮件列表发送要求订阅的邮件。请改用 Mailman 界面。

### 3.2 提交问题报告

建议、错误报告和代码贡献始终受到重视——请不要犹豫报告你发现的任何问题。附带修复的错误报告当然更受欢迎。

从具有互联网邮件连接的机器提交错误报告的首选方法是使用 [send-pr(1)](http://www.FreeBSD.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+8.1-RELEASE) 命令。以这种方式提交的“问题报告”（PR）将被归档并跟踪其进度；FreeBSD 开发者会尽最大努力尽快响应所有报告的错误。[所有活动 PR 的列表](http://www.FreeBSD.org/cgi/query-pr-summary.cgi) 可在 FreeBSD 网站上查阅；此列表有助于了解其他用户遇到的潜在问题。

请注意，[send-pr(1)](http://www.FreeBSD.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+8.1-RELEASE) 本身是一个 shell 脚本，应该易于移植到非 FreeBSD 系统。强烈推荐使用此接口。如果由于某种原因你无法使用 [send-pr(1)](http://www.FreeBSD.org/cgi/man.cgi?query=send-pr&sektion=1&manpath=FreeBSD+8.1-RELEASE) 提交错误报告，可以尝试将其发送至 [FreeBSD 问题报告邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-bugs)。

更多信息，请参阅 FreeBSD 网站上的 [“编写 FreeBSD 问题报告”](http://www.FreeBSD.org/doc/en_US.ISO8859-1/articles/problem-reports/)，其中包含许多关于编写和提交有效问题报告的有用提示。

## 4 进一步阅读

关于 FreeBSD 的信息来源很多；有些包含在本发行版中，有些则可在线或以印刷版本获取。

### 4.1 发行版文档

还有许多其他文件提供关于本发行版的更具体信息。这些文件以多种格式提供。大多数发行版会包含 ASCII 文本（`.TXT`）和 HTML（`.HTM`）两种格式。部分发行版还可能包含其他格式，如便携文档格式（`.PDF`）。

- **README.TXT**：本文件，提供关于 FreeBSD 的一般信息以及获取发行版的简要说明。

- **RELNOTES.TXT**：发行说明，展示 FreeBSD 8.1-RELEASE 相对上一版本（FreeBSD 8.0-RELEASE）的新增和变更内容。

- **HARDWARE.TXT**：硬件兼容列表，展示经过测试并已知可与 FreeBSD 配合工作的设备。

- **ERRATA.TXT**：发行版勘误。本文件包含发行后才发现的最新信息，主要适用于发行版（而非快照）。在安装 FreeBSD 发行版之前，务必查阅此文件，因为它包含自发行版创建以来已发现并修复的问题的最新信息。

在支持 [sysinstall(8)](http://www.FreeBSD.org/cgi/man.cgi?query=sysinstall&sektion=8&manpath=FreeBSD+8.1-RELEASE) 的平台（目前为 amd64、i386、ia64、pc98 和 sparc64）上，这些文档通常可在安装过程中通过文档菜单查阅。系统安装后，你可以通过重新运行 [sysinstall(8)](http://www.FreeBSD.org/cgi/man.cgi?query=sysinstall&sektion=8&manpath=FreeBSD+8.1-RELEASE) 工具重新访问此菜单。

>**注意**
>
> 在安装任何发行版之前，务必阅读其勘误，以了解任何“最新消息”或发行后的问题。随各发行版附带的勘误文件（很可能就在本文件旁边）按定义已经过时，但其他副本在互联网上保持更新，应作为本发行版的“当前勘误”查阅。这些其他勘误副本位于 <http://www.FreeBSD.org/releases/>（以及任何保持该位置最新镜像的站点）。

### 4.2 手册页

与几乎所有 UNIX 类操作系统一样，FreeBSD 附带一套联机手册页，可通过 [man(1)](http://www.FreeBSD.org/cgi/man.cgi?query=man&sektion=1&manpath=FreeBSD+8.1-RELEASE) 命令或 FreeBSD 网站上的 [超文本文档手册页网关](http://www.FreeBSD.org/cgi/man.cgi) 访问。通常，手册页提供关于 FreeBSD 用户可用的不同命令和 API 的信息。

在某些情况下，手册页是针对特定主题编写的。此类手册页的典型示例包括 [tuning(7)](http://www.freebsd.org/cgi/man.cgi?query=tuning&sektion=7&manpath=FreeBSD+8.1-RELEASE)（性能调优指南）、[security(7)](http://www.freebsd.org/cgi/man.cgi?query=security&sektion=7&manpath=FreeBSD+8.1-RELEASE)（FreeBSD 安全简介）和 [style(9)](http://www.freebsd.org/cgi/man.cgi?query=style&sektion=9&manpath=FreeBSD+8.1-RELEASE)（内核编码风格指南）。

### 4.3 书籍和文章

由 FreeBSD 项目维护的两套极具实用价值的 FreeBSD 相关信息合集是 FreeBSD 手册和 FreeBSD FAQ（常见问题文档）。[手册](http://www.FreeBSD.org/doc/en_US.ISO8859-1/books/handbook/) 和 [FAQ](http://www.FreeBSD.org/doc/en_US.ISO8859-1/books/faq/) 的在线版本始终可从 [FreeBSD 文档页面](http://www.FreeBSD.org/docs.html) 或其镜像获取。如果你安装了 `doc` 发行版，你可以使用 Web 浏览器在本地阅读手册和 FAQ。特别地，请注意手册包含 FreeBSD 安装的逐步指南。

由 FreeBSD 项目维护的许多在线书籍和文章涵盖更专业的 FreeBSD 相关主题。这些材料涵盖广泛的主题，从邮件列表的有效使用，到 FreeBSD 与其他操作系统的双启动，再到新提交者的指南。与手册和 FAQ 一样，这些文档可从 FreeBSD 文档页面或 `doc` 发行版中获取。

关于 FreeBSD 的其他书籍和文档列表见 FreeBSD 手册的 [参考书目](http://www.FreeBSD.org/doc/en_US.ISO8859-1/books/handbook/bibliography.html)。由于 FreeBSD 拥有深厚的 UNIX 传统，许多为 UNIX 系统编写的其他文章和书籍同样适用，其中一些也列在参考书目中。
