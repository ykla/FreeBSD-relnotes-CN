# FreeBSD 6.0-RELEASE 发行公告

**日期：** 2005 年 11 月 4 日 08:40:04 -0700
**发件人：** Scott Long <scottl@FreeBSD.org>
**收件人：** <freebsd-announce@FreeBSD.org>
**主题：** [FreeBSD-Announce] FreeBSD 6.0 Released

我非常荣幸地宣布 FreeBSD 6.0-RELEASE 现已发布。此版本是交付过去几年在 FreeBSD 5.x 系列中开发的高性能和企业级特性的下一步。自 5.4 以来的多项变更包括：

- 文件系统和直接磁盘访问层的显著性能改进。文件系统现在支持多线程，可充分利用多 CPU 系统。
- 扩展了对无线网卡的支持，并新增对 WPA 无线安全协议的支持。
- 对 PowerPC 平台的实验性支持。

有关新功能和已知问题的完整列表，请参见发行说明和勘误列表，地址如下：

- [http://www.FreeBSD.org/releases/6.0R/relnotes.html](http://www.freebsd.org/releases/6.0R/relnotes.html)
- [http://www.FreeBSD.org/releases/6.0R/errata.html](http://www.freebsd.org/releases/6.0R/errata.html)

有关 FreeBSD 发布工程活动的更多信息，请参见：

[http://www.FreeBSD.org/releng](http://www.freebsd.org/releng)

## 可用性

FreeBSD 6.0-RELEASE 支持 i386、pc98、alpha、sparc64、amd64、powerpc 和 ia64 架构，可使用可引导媒体直接通过网络安装或复制到本地 NFS/FTP 服务器。所有架构的发行版现已提供。

请通过从我们的支持供应商处购买介质来继续支持 FreeBSD 项目。以下公司将提供基于 FreeBSD 6.0 的产品：

- FreeBSD Mall, Inc. [http://www.freebsdmall.com/](http://www.freebsdmall.com/)
- Daemonnews, Inc. [http://www.bsdmall.com/freebsd1.html](http://www.bsdmall.com/freebsd1.html)

如果你无法负担介质形式的 FreeBSD、不耐烦等待，或仅想用于推广目的，那么请务必下载 ISO 镜像。我们无法保证所有镜像站点都提供较大的 ISO 镜像，但至少以下站点会提供。发行版镜像的 MD5 和 SHA256 校验和附在本消息末尾。
