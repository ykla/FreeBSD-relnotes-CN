# FreeBSD 2.0.5-ALPHA 发行公告

**日期**：1995 年 6 月 10 日（星期六）13:01:12 -0700
**发件人**：“Jordan K. Hubbard” <[jkh@freefall.cdrom.com](mailto:jkh@freefall.cdrom.com)>
**收件人**：<announce@freefall.cdrom.com>, <hackers@freefall.cdrom.com>
**主题**：宣布 FreeBSD 2.0.5 RELEASE！

像往常一样，我很荣幸（在某种程度上也松了口气！:)）地宣布发布 FreeBSD 2.0.5R——这是 2.0.5 系列的最终版本。

此版本既让我们一窥为 2.1R 计划的一些新技术和方向（希望令人兴奋），也提供了比 2.0R 更稳定、更易安装的替代方案。

此版本的亮点包括：

- 多语言文档文件。
- 完全菜单驱动的安装程序。
- 更多安装介质类型。
- 支持更广泛的 PC 硬件。
- 轻松挂载 DOS 分区，CD 设备自动挂载。
- “预制”安装类型，便于快速安装。
- 便捷的后续配置菜单

以及许多其他新功能和错误修复。

Ports 和软件包集合也已随 2.0.5R 一并发布，以避免同步错误。虽然这导致整体发行版体积增大，但至少能确保安装 Ports 和软件包时获得更一致的结果。

关于此版本的更多信息可参阅 RELNOTES 和 README 文件，因此我就不再赘述，留给你亲自去发现吧！

常规获取地址：

- [ftp://ftp.FreeBSD.org/pub/FreeBSD/2.0.5-RELEASE](ftp://ftp.FreeBSD.org/pub/FreeBSD/2.0.5-RELEASE)
- [ftp://freefall.cdrom.com/pub/FreeBSD/2.0.5-RELEASE](ftp://freefall.cdrom.com/pub/FreeBSD/2.0.5-RELEASE)

以及各镜像站点（在同步之后）。

Walnut Creek CDROM（本项目的主要赞助商）也提供 CD 版本。订购信息请参阅发行说明。

任何反馈请发送至 [hackers@FreeBSD.org](mailto:hackers@FreeBSD.org)。我将很快离境（约 3 小时后 :)），会尽量勤查邮件，但如需更快回复请发送至邮件列表。

感谢！

Jordan

**主题**：2.0.5-RELEASE 更新

没错，你猜对了。根据我们最近收到的关于原始 2.0.5-RELEASE 的反馈，我们进行了排查，在原始 `boot.flp` 镜像中发现了几个错误。现在 [ftp.cdrom.com](ftp://ftp.FreeBSD.org/pub/FreeBSD/2.0.5-RELEASE/UPDATES) 和 [freefall.cdrom.com](ftp://freefall.cdrom.com/pub/FreeBSD/2.0.5-RELEASE/UPDATES) 上的 **/pub/FreeBSD/2.0.5-RELEASE/UPDATES** 目录中提供了一个修正版的 `boot.flp`，可修复以下问题：

- 内核过大，无法在 4MB 内存的机器上启动。
- 可以指定文件系统挂载点，使得程序会在父文件系统挂载之前先挂载子文件系统（例如先挂载 **/usr/local** 再挂载 **/usr**）

由于 CD-ROM 尚未进入复制阶段，此更新后的软盘镜像也将出现在 CD-ROM 上。

向所有因第一套启动软盘遇到问题的用户致歉——下次发布版本时，我会亲自把 Jordan 钉在椅子上，再把椅子钉在他电脑前的地板上！

Gary
