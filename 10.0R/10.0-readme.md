# FreeBSD 10.0-RELEASE 自述文件

## 摘要

本文简要介绍 FreeBSD 10.0-RELEASE。包括如何获取 FreeBSD 的相关信息、联系 FreeBSD 项目的各种方式，以及其他信息来源的指引。

## 引言

本发行版是 FreeBSD 10.0-RELEASE，位于 10.0-STABLE 分支的最新节点。

## 关于 FreeBSD

FreeBSD 是一款基于 4.4 BSD Lite 的操作系统，适用于 AMD64 和 Intel EM64T 架构的 PC 硬件（amd64）、Intel、AMD、Cyrix 或 NexGen “x86”架构的 PC 硬件（i386）、Intel Itanium 处理器架构的计算机（ia64）、NEC PC-9801/9821 系列个人电脑及兼容机（pc98）以及 UltraSPARC 机器（sparc64）。
ARM（arm）、MIPS（mips）和 PowerPC（powerpc）架构的版本目前也在开发中。
FreeBSD 兼容各种外设和配置，可用于从软件开发、游戏到互联网服务提供的各类用途。

本 FreeBSD 发行版包含运行此类系统所需的全部内容，包括内核和基本系统中所有实用工具的完整源代码。
安装源代码发行版后，可通过一条命令从头重新编译整个系统，这使其成为学生、研究人员或单纯想了解系统工作原理的用户的理想选择。

本项目还提供大量第三方移植软件集合（Ports），便于获取和安装所有你喜爱的传统 UNIX 实用工具。
每个 Port 由一组脚本组成，用于检索、配置、构建和安装某个软件，只需一条命令即可完成。
超过 24,000 个 Port，从编辑器到编程语言再到图形应用程序，使 FreeBSD 成为功能强大且全面的操作环境，远超许多商业版 UNIX 所提供的功能。
大多数 Port 还提供预编译的软件包，可通过安装程序快速安装。

## 目标受众

本 FreeBSD 发行版适合所有用户。它经过了一段时间的测试和质量保证检查，以确保最高的可靠性和稳定性。

## 获取 FreeBSD

FreeBSD 可通过多种方式获取。
本节重点介绍主要用于获取完整 FreeBSD 发行版的方式，而非更新现有安装。

### CDROM 和 DVD

FreeBSD -RELEASE 发行版可从多家出版商订购 CDROM 或 DVD。
这通常是新安装时获取 FreeBSD 最便捷的方式，因为它提供了一种在必要时快速重装系统的便利方法。
部分发行版包含 FreeBSD Ports 中的一些可选预编译软件包，或其他附加材料。

项目已知的 CDROM 和 DVD 出版商列表见手册的 [获取 FreeBSD](https://docs.freebsd.org/en/books/handbook/mirrors/) 附录。

### FTP

可使用 FTP 从 `ftp://ftp.FreeBSD.org/`（FreeBSD 官方发行站点）或其任何镜像站点检索 FreeBSD 及其全部可选软件包。

镜像 FreeBSD 的站点列表见手册的 [FTP 站点](https://docs.freebsd.org/en/books/handbook/mirrors/#ftp) 一节。
强烈建议从网络距离较近的镜像下载发行版。

始终欢迎增加新的镜像站点。
成为官方镜像站点的更多信息，请联系 `<freebsd-admin@FreeBSD.org>`。
镜像站点的实用信息也可参阅 [镜像 FreeBSD](https://docs.freebsd.org/en/articles/hubs/) 文章。

镜像站点通常包含用于创建 FreeBSD 发行版 CDROM 的 ISO 镜像，通常还包含软盘镜像（适用于支持的平台），以及通过网络安装所需的文件。
最后，镜像站点通常还包含最新发行版的一组软件包。

## 联系 FreeBSD 项目

### 电子邮件和邮件列表

如有一般问题或技术支持，请联系 [freebsd-questions 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-questions)。

如果你跟踪 `10.0-STABLE` 开发进展，*必须* 加入 [freebsd-stable 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-stable)，以了解可能影响系统使用和维护的最新开发和变更动态。

作为主要由志愿者推动的项目，FreeBSD 项目始终欢迎愿意提供帮助的人手——期望实现的增强功能远多于实现这些功能所拥有的时间。
如需就技术问题联系开发者或提供帮助，请联系 [freebsd-hackers 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-hackers)。

上述邮件列表的流量可能较大。
如果你的邮件访问速度较慢或费用较高，或只关注重大公告，可订阅 [freebsd-announce 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-announce)。

所有公共邮件列表均可供任何希望加入的人订阅。
访问 [FreeBSD Mailman 信息页面](https://lists.freebsd.org/mailman/listinfo)，可获取关于加入各列表、访问归档等的更多信息。
此外还有许多面向特定兴趣小组的邮件列表此处未提及；更多信息可从 Mailman 页面或 FreeBSD 网站的 [邮件列表](https://www.freebsd.org/support/#mailing-list) 部分获取。

> **重要**：
>
> 请勿向邮件列表发送订阅请求。请使用 Mailman 界面。

### 提交问题报告

欢迎提出建议、提交缺陷报告和贡献代码。
如发现任何问题，请不要犹豫，及时报告。
附带修复的缺陷报告当然更受欢迎。

从具有 Internet 连接的机器提交缺陷报告的首选方式是使用 [send-pr(1)](https://man.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&format=html) 命令。
以此方式提交的问题报告（PR）将被归档并跟踪其处理进度；FreeBSD 开发者将尽最大努力尽快响应所有报告的缺陷。
FreeBSD 网站上提供 [所有活跃 PR 的列表](https://www.freebsd.org/cgi/query-pr-summary.cgi)；该列表有助于了解其他用户遇到的潜在问题。

请注意，[send-pr(1)](https://man.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&format=html) 本身是一个 shell 脚本，即使是非 FreeBSD 系统也应易于移植。
强烈推荐使用此接口。
如果由于某些原因无法使用 [send-pr(1)](https://man.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&format=html) 提交缺陷报告，可以尝试发送至 [freebsd-bugs 邮件列表](https://lists.freebsd.org/mailman/listinfo/freebsd-bugs)。

更多信息，请参阅 FreeBSD 网站上的 [编写 FreeBSD 问题报告](https://docs.freebsd.org/en/articles/problem-reports/)，其中包含编写和提交有效问题报告的诸多实用提示。

## 延伸阅读

关于 FreeBSD 的信息来源很多；部分包含在本发行版中，其他则可在线获取或以印刷版本获取。

### 发行版文档

若干其他文件提供了关于本发行版的更具体信息。
这些文件以多种格式提供。
大多数发行版将同时包含 ASCII 文本（`.TXT`）和 HTML（`.HTM`）版本。
部分发行版还可能包含其他格式，如便携文档格式（`.PDF`）。

* `README.TXT`：本文件，提供关于 FreeBSD 的一般信息以及获取发行版的简要说明。
* `RELNOTES.TXT`：发行说明，展示 FreeBSD 10.0-RELEASE 相对于前一版本（FreeBSD 9.2-RELEASE）的新增和不同之处。
* `HARDWARE.TXT`：硬件兼容列表，展示已测试并已知可与 FreeBSD 配合使用的设备。
* `ERRATA.TXT`：发行版勘误。本文件包含发布后期及发布后的最新信息，主要适用于发行版（而非快照）。在安装 FreeBSD 发行版之前，务必查阅本文件，因为它包含自发行版创建以来发现并修复的问题的最新信息。

在支持 [bsdinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=bsdinstall&sektion=8&format=html) 的平台（目前为 amd64、i386、ia64、pc98 和 sparc64）上，这些文档通常可在安装过程中通过文档菜单访问。
系统安装完成后，可重新运行 [bsdinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=bsdinstall&sektion=8&format=html) 工具再次访问此菜单。

> **注意**：
>
> 要了解任何最新消息或发布后的问题，在安装任何发行版之前，阅读其勘误文件极为重要。
> 随各发行版附带的勘误文件（很可能就在本文件旁边）按定义已经过时，但其他副本在互联网上保持更新，应作为本发行版的当前勘误查阅。
> 这些其他勘误副本位于 `https://www.freebsd.org/releases/`（以及任何保持该位置最新镜像的站点）。

### 手册页

与几乎所有 UNIX 类操作系统一样，FreeBSD 包含一套联机手册页，可通过 [man(1)](https://man.freebsd.org/cgi/man.cgi?query=man&sektion=1&format=html) 命令或 FreeBSD 网站上的 [超文本手册页网关](https://man.freebsd.org) 访问。
通常，手册页提供关于 FreeBSD 用户可用的不同命令和 API 的信息。

部分手册页旨在提供特定主题的信息。
值得注意的例子包括 [tuning(7)](https://man.freebsd.org/cgi/man.cgi?query=tuning&sektion=7&format=html)（性能调优指南）、[security(7)](https://man.freebsd.org/cgi/man.cgi?query=security&sektion=7&format=html)（FreeBSD 安全简介）和 [style(9)](https://man.freebsd.org/cgi/man.cgi?query=style&sektion=9&format=html)（内核编码风格指南）。

### 书籍和文章

FreeBSD 项目维护的两个非常实用的 FreeBSD 相关信息集合是 *FreeBSD 手册* 和 *FreeBSD FAQ*（常见问题解答）。
[手册](https://docs.freebsd.org/en/books/handbook/) 和 [FAQ](https://docs.freebsd.org/en/books/faq/) 的在线版本可从 [FreeBSD 文档页面](https://docs.freebsd.org/) 或其镜像获取。
如果安装 `doc` 发行版，可使用 Web 浏览器本地阅读手册和 FAQ。
特别地，手册包含 FreeBSD 安装的分步指南。

FreeBSD 项目维护的若干联机书籍和文章涵盖更专门的 FreeBSD 相关主题。
这些材料涵盖广泛的主题，从邮件列表的有效使用，到 FreeBSD 与其他操作系统的双启动，再到新提交者的指南。
与手册和 FAQ 一样，这些文档可从 FreeBSD 文档页面或 `doc` 发行版获取。

关于 FreeBSD 的其他书籍和文档列表可在 FreeBSD 手册的 [参考文献](https://docs.freebsd.org/en/books/handbook/bibliography/) 中找到。
由于 FreeBSD 深厚的 UNIX 血统，许多为 UNIX 系统编写的其他文章和书籍同样适用，其中部分也列于参考文献中。
