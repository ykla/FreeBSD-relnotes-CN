# FreeBSD 10.0-RELEASE 发行公告

日期：2014 年 1 月 20 日

FreeBSD 发行工程团队很高兴地宣布 FreeBSD 10.0-RELEASE 现已可用。这是 stable/10 分支的第一个发行版本。

部分亮点：

* 在以 [clang(1)](https://man.freebsd.org/cgi/man.cgi?query=clang&sektion=1&format=html) 作为默认编译器的架构上，GCC 不再默认安装。
* Unbound 已作为本地缓存 DNS 解析器导入到基本系统。
* BIND 已从基本系统中移除。
* [make(1)](https://man.freebsd.org/cgi/man.cgi?query=make&sektion=1&format=html) 已被来自 NetBSD 项目的 bmake(1) 取代。
* [pkg(7)](https://man.freebsd.org/cgi/man.cgi?query=pkg&sektion=7&format=html) 现在是默认的软件包管理工具。
* **pkg_add(1)**、**pkg_delete(1)** 及相关工具已被移除。
* 虚拟化方面的重大增强，包括新增 [bhyve(8)](https://man.freebsd.org/cgi/man.cgi?query=bhyve&sektion=8&format=html)、[virtio(4)](https://man.freebsd.org/cgi/man.cgi?query=virtio&sektion=4&format=html)，以及提供 FreeBSD 作为 Microsoft Hyper-V 客户机操作系统支持的原生半虚拟化驱动程序。
* ZFS 新增对固态硬盘的 TRIM 支持。
* ZFS 新增高性能 LZ4 压缩算法支持。

关于新功能和已知问题的完整列表，请参阅在线发行说明和勘误列表，地址如下：

* <https://www.FreeBSD.org/releases/10.0R/errata/>

## 可用性

FreeBSD 10.0-RELEASE 现已可用于 amd64、i386、ia64、powerpc、powerpc64 和 sparc64 架构。

FreeBSD 10.0 可以从可引导 ISO 镜像或通过网络安装。某些架构还支持从 USB 闪存盘安装。所需文件可通过 FTP 下载，如下节所述。虽然一些较小的 FTP 镜像可能不包含所有架构，但它们通常都会包含较常见的架构，如 amd64 和 i386。

发行版 ISO 和闪存盘镜像的 SHA256 和 MD5 校验和附于本消息底部。

本公告的 PGP 签名版本可从以下地址获取：

* <https://www.FreeBSD.org/releases/10.0R/announce.asc>

发行版中所提供镜像的用途如下：

*dvd1*：
包含安装 FreeBSD 基本系统、文档以及一组旨在让图形工作站快速运行起来的预构建软件包所需的全部内容。它还支持引导进入基于“livefs”的救援模式。如果你能刻录并使用 DVD 大小的介质，这应该就是你所需的全部。

*disc1*：
包含 FreeBSD 基本系统。它还支持引导进入基于“livefs”的救援模式。不包含预构建软件包。

*bootonly*：
支持使用 CDROM 驱动器引导机器，但不包含从 CD 本身安装 FreeBSD 的安装发行版组件。你需要从 CD 引导后执行基于网络的安装（例如从 FTP 服务器）。

*memstick*：
可写入 USB 闪存盘（U 盘），用于在能够从 USB 驱动器引导的机器上执行安装。它还支持引导进入基于“livefs”的救援模式。不包含预构建软件包。

作为使用 memstick 镜像的一个示例，假设 USB 驱动器在你的机器上显示为 **/dev/da0**，类似如下命令应该可行：

```sh
# dd if=FreeBSD-10.0-RELEASE-amd64-memstick.img of=/dev/da0 bs=10240 conv=sync
```

务必确保目标（of=）正确。

## FTP

FreeBSD 10.0-RELEASE 可从以下站点通过 ftp 下载：

* `ftp://ftp.freebsd.org/pub/FreeBSD/releases/ISO-IMAGES/10.0/`

但在尝试此站点之前，请先通过以下方式检查你所在地区的镜像：

* `ftp://ftp.<yourdomain>.FreeBSD.org/pub/FreeBSD`

任何额外的镜像站点将标记为 ftp2、ftp3 等。

关于 FreeBSD 镜像站点的更多信息可从以下地址获取：

* <https://docs.freebsd.org/en/books/handbook/mirrors/#mirrors-ftp>

关于安装 FreeBSD 或将现有机器升级至 10.0-RELEASE 的说明，请参阅：

* <https://www.FreeBSD.org/releases/10.0R/installation/>
