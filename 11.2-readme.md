# FreeBSD 11.2-RELEASE 自述文件

## 摘要

本文简要介绍 FreeBSD 11.2-RELEASE。包括如何获取 FreeBSD、联系 FreeBSD 项目的各种方式，以及指向其他信息来源的指针。

## 引言

本发行版是 FreeBSD 11.2-RELEASE，是 11.2-STABLE 分支上的最新点版本。

## 关于 FreeBSD

FreeBSD 是一个基于 4.4 BSD Lite 的操作系统，适用于 AMD64 和 Intel EM64T 架构的 PC 硬件（amd64）、Intel、AMD、Cyrix 或 NexGen“x86”PC 硬件（i386）、NEC PC-9801/9821 系列 PC 及兼容机（pc98），以及 UltraSPARC(R) 机器（sparc64）。针对 ARM(R)（arm）、MIPS(R)（mips）和 PowerPC(R)（powerpc）架构的版本也在开发中。FreeBSD 兼容各种外设和配置，可用于从软件开发、游戏到互联网服务提供的各种用途。

本 FreeBSD 发行版包含运行此类系统所需的一切，包括内核和基本发行版中所有实用程序的完整源代码。安装源代码发行版后，你可以通过一条命令从零开始重新编译整个系统，使其成为学生、研究人员或只是想了解其工作原理的用户的理想选择。

还提供了大量第三方移植软件（“Ports”），便于获取和安装所有你喜爱的传统 UNIX(R) 实用程序。每个“port”由一组脚本组成，可通过一条命令检索、配置、构建和安装某个软件。超过 24,000 个 port，从编辑器到编程语言再到图形应用程序，使 FreeBSD 成为一个强大而全面的操作环境，远超许多商业 UNIX(R) 版本所提供的。大多数 port 也提供预编译的“包”，可从安装程序快速安装。

## 目标受众

本 FreeBSD 发行版适用于所有用户。它经过了一段测试和质量保证检查期，以确保最高的可靠性和可信度。

## 获取 FreeBSD

可通过多种方式获取 FreeBSD。本节重点介绍主要用于获取完整 FreeBSD 发行版的方式，而非更新现有安装。

### CDROM 和 DVD

可从多家发行商订购 CDROM 或 DVD 上的 FreeBSD -RELEASE 发行版。这通常是获取 FreeBSD 进行新安装的最便捷方式，因为它提供了在必要时快速重新安装系统的便利方式。某些发行版包含 FreeBSD Ports 中的可选预编译“包”，或其他额外材料。

项目已知的 CDROM 和 DVD 发行商列表见手册的 [获取 FreeBSD](https://docs.freebsd.org/en/books/handbook/#mirrors) 附录。

### HTTPS

可从 <https://download.FreeBSD.org/>（官方 FreeBSD 发行站点）通过 HTTPS 下载 FreeBSD 发行版。

### FTP

可使用 FTP 从 `ftp://ftp.FreeBSD.org/` 或其任何“镜像”检索 FreeBSD 及其任何或所有可选包。

镜像 FreeBSD 的位置列表见手册的 [FTP 站点](https://docs.freebsd.org/en/books/handbook/#mirrors-ftp) 部分。强烈建议从网络距离较近的镜像下载发行版。

始终欢迎增加镜像站点。联系 `<freebsd-admin@FreeBSD.org>` 获取有关成为官方镜像站点的更多详细信息。你还可以在 [镜像 FreeBSD](https://docs.freebsd.org/en/books/handbook/#mirrors) 文章中找到镜像站点的有用信息。

镜像通常包含用于创建 FreeBSD 发行版 CDROM 的 ISO 镜像。它们通常还包含软盘镜像（适用于适用的平台），以及通过网络进行安装所需的文件。最后，镜像站点通常包含最新发行版的一组包。

## 联系 FreeBSD 项目

### 电子邮件和邮件列表

如有任何问题或一般技术支持问题，请发送邮件至 [FreeBSD 一般问题邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-questions)。

如果你正在跟踪 11.2-STABLE 开发工作，_必须_ 加入 [FreeBSD-STABLE 邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-current)，以了解可能影响你使用和维护系统的最新开发和变更。

作为一个主要由志愿者推动的项目，FreeBSD 项目始终欢迎愿意提供帮助的志愿者——已经有远超时间所能实现的期望增强。要就技术事项联系开发者或提供帮助，请发送邮件至 [FreeBSD 技术讨论邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-hackers)。

请注意，这些邮件列表可能会产生 _大量_ 流量。如果你的邮件访问缓慢或昂贵，或仅希望了解 FreeBSD 的重大事件，可能更倾向于订阅 [FreeBSD 公告邮件列表](http://lists.FreeBSD.org/mailman/listinfo/freebsd-announce)。

所有邮件列表均可由任何愿意加入的人自由加入。访问 [FreeBSD Mailman 信息页面](https://lists.freebsd.org/mailman/listinfo)。这将提供有关加入各种列表、访问归档等的更多信息。还有许多针对此处未提及的特定兴趣组的邮件列表；可从 Mailman 页面或 FreeBSD 网站的 [邮件列表部分](https://www.freebsd.org/support/#mailing-list) 获取更多信息。

> **重要**：
>
> _不要_ 向列表发送要求订阅的邮件。请改用 Mailman 界面。

### 提交问题报告

建议、错误报告和代码贡献始终受到重视——请随时报告你发现的任何问题。附有修复的错误报告当然更受欢迎。

从具有互联网连接的机器提交错误报告的首选方法是使用 Bugzilla 错误跟踪器。以这种方式提交的“问题报告”（PR）将被归档并跟踪进度；FreeBSD 开发者将尽最大努力尽快响应所有报告的错误。[所有活动 PR 列表](https://bugs.FreeBSD.org/search/) 可在 FreeBSD 网站上找到；此列表有助于了解其他用户遇到的潜在问题。

请注意，[send-pr(1)](https://man.freebsd.org/cgi/man.cgi?query=send-pr&sektion=1&format=html) 已弃用。

更多信息，请参阅 FreeBSD 网站上的 [《编写 FreeBSD 问题报告》](https://docs.freebsd.org/en/articles/problem-reports/)，其中包含许多关于编写和提交有效问题报告的有用提示。

## 进一步阅读

关于 FreeBSD 的信息来源很多；一些包含在本发行版中，而其他可在线或在印刷版本中获取。

### 发行版文档

许多其他文件提供关于本发行版的更具体信息。这些文件以各种格式提供。大多数发行版包含 ASCII 文本（`.TXT`）和 HTML（`.HTM`）两种版本。某些发行版还可能包含其他格式，如可移植文档格式（`.PDF`）。

* `README.TXT`：本文件，提供关于 FreeBSD 的一些一般信息以及获取发行版的一些简要说明。
* `RELNOTES.TXT`：发行说明，展示 FreeBSD 11.2-RELEASE 相对于上一版本（FreeBSD 11.1-RELEASE）的新功能和差异。
* `HARDWARE.TXT`：硬件兼容列表，展示已测试 FreeBSD 并已知可工作的设备。
* `ERRATA.TXT`：发行版勘误。本文件包含晚期、发行后信息，主要适用于发行版（而非快照）。在安装 FreeBSD 发行版之前查阅本文件非常重要，因为它包含自发行版创建以来发现并修复的问题的最新信息。

在支持 [bsdinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=bsdinstall&sektion=8&format=html) 的平台上（目前为 amd64、i386、ia64、pc98 和 sparc64），这些文档通常可在安装期间通过文档菜单获取。系统安装后，可通过重新运行 [bsdinstall(8)](https://man.freebsd.org/cgi/man.cgi?query=bsdinstall&sektion=8&format=html) 实用程序重新访问此菜单。

> **注意**：
>
> 在安装任何给定发行版之前，阅读其勘误以了解任何“最新消息”或发行后问题至关重要。随每个发行版附带的勘误文件（很可能就在本文件旁边）按定义已经过时，但其他副本在互联网上保持更新，应作为本发行版的“当前勘误”查阅。这些其他勘误副本位于 FreeBSD 11.2-RELEASE 页面（以及任何保持该位置最新镜像的站点）。

### 手册页

与几乎所有类 UNIX(R) 操作系统一样，FreeBSD 附带一组联机手册页，可通过 [man(1)](https://man.freebsd.org/cgi/man.cgi?query=man&sektion=1&format=html) 命令或 FreeBSD 网站上的 [超文本手册页网关](https://man.freebsd.org/cgi/man.cgi) 访问。通常，手册页提供有关 FreeBSD 用户可用的不同命令和 API 的信息。

在某些情况下，手册页是为特定主题编写的。此类手册页的 notable 示例包括 [tuning(7)](https://man.freebsd.org/cgi/man.cgi?query=tuning&sektion=7&format=html)（性能调优指南）、[security(7)](https://man.freebsd.org/cgi/man.cgi?query=security&sektion=7&format=html)（FreeBSD 安全简介）和 [style(9)](https://man.freebsd.org/cgi/man.cgi?query=style&sektion=9&format=html)（内核编码风格指南）。

### 书籍和文章

由 FreeBSD 项目维护的两个非常实用的 FreeBSD 相关信息集合是 FreeBSD 手册和 FreeBSD FAQ（常见问题解答）。[手册](https://docs.freebsd.org/en/books/handbook/) 和 [FAQ](https://docs.freebsd.org/en/books/faq/) 的在线版本始终可从 [FreeBSD 文档页面](https://docs.freebsd.org/) 或其镜像获取。如果安装 `doc` 发行版集，可使用 Web 浏览器在本地阅读手册和 FAQ。特别地，手册包含安装 FreeBSD 的分步指南。

由 FreeBSD 项目维护的多本在线书籍和文章涵盖更专业的 FreeBSD 相关主题。这些材料涵盖广泛的主题，从有效使用邮件列表，到 FreeBSD 与其他操作系统双启动，到新提交者的指南。与手册和 FAQ 一样，这些文档可从 FreeBSD 文档页面或 `doc` 发行版集获取。

关于 FreeBSD 的其他书籍和文档列表见 FreeBSD 手册的 [参考书目](https://docs.freebsd.org/en/books/handbook/#bibliography)。由于 FreeBSD 强大的 UNIX(R) 血统，许多为 UNIX(R) 系统编写的其他文章和书籍同样适用，其中一些也在参考书目中列出。
