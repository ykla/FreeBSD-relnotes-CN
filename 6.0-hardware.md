# FreeBSD 6.0-RELEASE 硬件兼容列表

FreeBSD 的硬件说明针对不同平台进行了定制，因为许多设备仅在特定处理器或架构上受支持（或仅与特定处理器或架构相关）。

FreeBSD 6.0-RELEASE 的硬件说明可在以下平台获取：

- [alpha](6.0-alpha-hardware.md)
- [amd64](6.0-amd64-hardware.md)
- [i386](6.0-i386-hardware.md)
- [ia64](6.0-ia64-hardware.md)
- [pc98](6.0-pc98-hardware.md)
- [sparc64](6.0-sparc64-hardware.md)

当前正在开发的所有平台列表可在 [支持的平台](https://www.freebsd.org/platforms/) 页面上找到。
