# FreeBSD 3.5-RELEASE 发行公告

**日期：** Sat, 24 Jun 2000 18:23:01 -0700
**发件人：** “Jordan K. Hubbard” <jkh@freebsd.org>
**主题：** FreeBSD 3.5 now available for x86

我很高兴地宣布 FreeBSD 3.5-RELEASE 已发布，这是 3.x-STABLE 分支技术的**最后**一个版本。继 1999 年 12 月 FreeBSD 3.4 发布之后，修复了许多错误，处理了重要的安全问题，甚至还添加了一些新功能。更多信息请参见[发行说明](../notes)。

FreeBSD 3.5-RELEASE 可从 [ftp.FreeBSD.org](ftp://ftp.FreeBSD.org/pub/FreeBSD/) 以及全球各地的 [FTP 镜像站点](https://www.FreeBSD.org/handbook/mirrors.html) 获取。也可向 [The FreeBSD Mall](http://www.FreeBSDMall.com/) 订购 CD，不久后即可发货，为 4 CD 套装，内含 x86 架构的安装文件，以及大量对程序员和终端用户都颇具价值的其他资料。

我们无法保证所有镜像站点都会提供这个相当大的安装镜像（660MB），但至少可从以下地址获取：

**ftp://ftp.FreeBSD.org/pub/FreeBSD/releases/i386/ISO-IMAGES/3.5-install.iso**

以及更传统的 3.5-RELEASE 安装文件。如果你买不起 CD、等不及，或只是想用于推广目的，那么尽管下载 ISO；否则请继续从 BSDi 购买其官方 CD 发行版来支持 FreeBSD 项目。

FreeBSD 的官方 FTP 发行站点为：

在尝试访问中央 FTP 站点之前，请先前往以下地址查看你所在区域的镜像站点：

**ftp://ftp.<yourdomain>.freebsd.org/pub/FreeBSD**

任何额外的镜像站点将依次命名为 ftp2、ftp3 等。

FreeBSD（2.0C 及以后版本）受出口限制代码（eBones 和 secure）的最新版本也可从以下位置获取。既然 FreeBSD 已获得美国政府出口加密代码的许可，你可以从这些位置或 ftp.freebsd.org 获取。
